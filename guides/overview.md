# 深度綜述 · Deep Guides

單頁解構回答「這一個方法怎麼拆」。**綜述回答「這一族到底在賭什麼」**——把一個方法族的幾十頁串成一條演進主線、幾條核心張力、一份失效模式合集，再給不同讀者一條有序讀法。

每篇都讀過該族數十頁真實解構後寫成，論斷可追溯到具體頁面；數字一律逐字引自原頁，缺的寫「未披露」。

| 綜述 | 頁數底盤 | 它替你回答的問題 |
|---|--:|---|
| [時序基礎模型全景](time-series-forecasting.md) | 87 | 一個在電力/氣象上刷穿榜的通用時序模型，憑什麼在低信噪比、非平穩的金融價格上生出 Alpha？**什麼移得動、什麼移不動。** |
| [因子挖掘全景](factor-mining.md) | 61 | 從手工 alpha 到 LLM 自動因子生產線——真正的護城河不是更高 IC 的單因子，而是**發現→過濾→組合**這條線，與它抗衰減的能力。 |
| [LLM-Agentic 交易全景](llm-agentic.md) | 50 | LLM 進交易到底加了什麼真價值、什麼只是把幻覺與 look-ahead 洩漏包裝成新架構？**哪幾層 alpha 是真的。** |
| [執行與組合 RL 全景](reinforcement-learning.md) | 39 | 把交易寫成 MDP 值不值？RL 的真戰場是**執行/對沖/路由**這類有即時反饋的子問題，而非端到端選股。 |
| [組合優化全景](portfolio-optimization.md) | 34 | 護城河不是更準的收益預測，而是**把預測誤差關進籠子的那道工序**——協方差怎麼估、換手怎麼壓、決策目標怎麼直接回傳梯度。 |
| [市場微結構全景](market-microstructure.md) | 26 | 毫秒到日內的 LOB 與訂單流如何形成價格。核心只有一句：扣掉延遲與成本，**這條 alpha 你來得及吃嗎**。 |
| [自動化研究迴圈建造指南](autoresearch-loop.md) | — | 唯一一篇不講方法族、講**研究流程**的：把 loop engineering 接上 trial accounting。核心只有一句：每跑一輪迴圈，你的統計顯著性就少一分——**這筆帳不記，你會拿到一張漂亮的偽發現證書**。 |

> 讀法：先讀跟你當前問題最近的一篇的「核心張力」與「什麼在持續有效 vs 什麼被擁擠掉」兩節，再順著該篇的 reading path 進單頁解構。
