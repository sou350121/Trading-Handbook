# 盈亏平衡夏普比率 解構

**TL;DR:** ① 推導多因子模型樣本內/外夏普比率的精確有限樣本分佈，量化估計風險對樣本外表現的懲罰。② 核心 trick 是引入預期缺口（Expected Shortfall）定義「盈虧平衡夏普比率」，作為模型比較的校準閾值。③ 對因子挖掘軸★：打破「因子越多/樣本內夏普越高越好」的幻覺，提供實盤前必須通過的理論門檻。④ 關鍵實證數字：當 N=2, T=60, c=5% 時，雙因子模型的總體夏普需比 CAPM 高 62% 方可視為等效。

**X-Ray.** 本文將資產定價模型的比較從「漸近正態假設下的點估計」強制拉回「有限樣本下的隨機變量比較」。傳統因子回測常忽略參數估計誤差在樣本外階段的非對稱放大效應（左偏分佈），導致過度擬合的複雜模型（高 N）在實盤中遭遇斷崖式回撤。該框架不依賴交易成本或流動性摩擦假設，純粹從統計學角度證明：樣本內夏普是總體夏普的向上偏誤估計量，而樣本外夏普恆低於總體值。對量化研究員而言，這意味著任何未經過「盈虧平衡閾值」校準的因子疊加都是危險的。該方法打不開的 envelope 在於其嚴格依賴 i.i.d. 多元正態假設；若實盤數據呈現厚尾或 regime switching，估計風險的實際懲罰將遠超理論下限。它不是預測模型，而是模型篩選的「滅火器」，強制研究員在提交 alpha 前計算其統計學上的生存閾值。

## §1 · 架構 / Core Mechanism
**1.1 三大改動 vs 前作**
| 維度 | 傳統模型比較 (GRS/貝葉斯) | 本文方法 (盈虧平衡夏普比率) |
|---|---|---|
| 比較基準 | 總體夏普比率相等 ($\theta_N = \theta_1$) | 樣本外夏普比率分佈校準 ($\theta_N \ge \theta_{BE}$) |
| 估計風險處理 | 忽略或依賴大樣本漸近收斂 | 精確有限樣本分佈 + 預期缺口度量 |
| 模型複雜度懲罰 | 無明確量化閾值 | 隨因子數 $N$ 增加/窗口 $T$ 縮短而動態抬升 |

**1.2 ⚡ Eureka 一句話 trick + 直覺**
用預期缺口（Expected Shortfall）錨定樣本外夏普分佈的左尾，將「模型是否優於 CAPM」的假設檢驗，轉化為「總體夏普需達到何種閾值才能覆蓋估計風險」的確定性計算。

**1.3 信息流 ASCII 圖**
```
歷史因子收益 (T x N)
      │
      ▼
估計窗口 → 計算樣本最優權重 (w_hat)
      │
      ├──▶ 樣本內夏普 (IS-SR) → 向上偏誤估計量 (不可實盤兌現)
      │
      └──▶ 樣本外期間 → 應用 w_hat → 樣本外夏普 (OS-SR)
                │
                ▼
        有限樣本分佈 (高度左偏)
                │
                ▼
        預期缺口 (c%) → 盈虧平衡夏普比率 (θ_BE)
                │
                ▼
        模型比較閾值 (θ_N ≥ θ_BE ? 通過/淘汰)
```

## §2 · 數學層
📌 **Napkin Formula**：
$OS\text{-}SR \sim \text{Distribution}(N, T, \theta)$
$\theta_{BE} \text{ s.t. } P(OS\text{-}SR < \theta_1) = c$
複雜度：$O(T \times N^2)$ 矩陣求逆與分佈推導。
直覺：樣本內夏普恆大於總體夏普，樣本外夏普恆小於總體夏普。兩者的差距與樣本內夏普呈負相關（觀察值越高，樣本外失望越大）。分佈嚴重左偏，故用預期缺口而非均值定義閾值。
Loss/訓練細節：無傳統 ML loss。基於 i.i.d. 多元正態假設下的隨機表示（Random Representation）推導精確分佈，替代傳統 GRS 檢驗的 F 分佈假設。

## §3 · 數據層
資料規模/頻率/市場/時段：1967 年 1 月至 2021 年 12 月月度數據。市場為美國股票市場（價值加權市場投資組合為基準）。樣本外與容量假設：將樣本期分為兩半（前半估計窗口，後半樣本外期間）。容量假設隱含於有限樣本分佈的 $N$ 與 $T$ 參數中，未披露具體資產池橫截面規模，僅聚焦可交易因子組合。

**解讀**：表格中的 Δ 欄多為「未披露」，因導讀僅提供單點實證數值，未給出傳統方法在相同設定下的對照組數據。46.97% 至 71.28% 的樣本外衰減是真實的統計學現象，非過擬合單一產物，而是估計風險在有限樣本下的必然數學結果。62% 的閾值抬升揭示了複雜模型（高 N）的隱形成本：在傳統 GRS 檢驗中顯著優於 CAPM 的模型（如 BAB、FF-5），納入盈虧平衡校準後可能失去統計顯著性。此 Δ 代表的是「估計風險懲罰」，而非模型本身的預測能力增益。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations**：理論分析基於因子回報的 i.i.d. 多元正態分佈假設。若回報分佈具有厚尾特徵，估計風險問題可能會比正態分佈情況下更嚴重，因此結果僅為估計風險影響的下限。
**6.2 推斷的隱含假設**：Regime 依賴（假設機會集恆定，忽略宏觀週期切換）；容量（未考慮多空策略的賣空限制與交易成本）；數據泄漏（嚴格區分估計窗口與樣本外期間，但實盤中滾動窗口會引入重疊偏差）；Survivorship（使用全樣本期數據，未明確說明是否處理退市股票偏差）。

🎤 **Interview Tip**
正確答：「盈虧平衡夏普比率不是預測指標，而是模型篩選的統計學門檻。它通過預期缺口量化估計風險對樣本外表現的左偏懲罰，強制要求高維因子模型必須具備更高的總體夏普才能與低維基準等效。」
錯答：「它是一個能直接預測樣本外夏普的機器學習模型，或者它通過加入交易成本來修正夏普比率。」

**7.1 可證偽預測帶日期**：若 2026 年底前，實盤機構在嚴格控制估計窗口與樣本外劃分的前提下，仍普遍採用未經過盈虧平衡閾值校準的高 N 因子模型，且其樣本外夏普衰減幅度顯著低於 46.97%（導讀給出的 HMXZ 衰減下限），則該理論框架的 i.i.d. 正態假設或有限樣本分佈推導存在實證失效。

## §8 · For the Reader
- **因子研究員**：在提交新 alpha 前，務必計算其盈虧平衡閾值。若模型總體夏普未達標，直接淘汰，避免將回測幻覺帶入實盤。
- **組合配置/風控**：將此閾值嵌入因子池准入規則。對高 N 模型設置更嚴格的樣本外驗證期，對沖估計風險帶來的左尾回撤。
- **研究學生/QuantML 愛好者**：跳過傳統 GRS 檢驗的漸近假設，直接複現有限樣本分佈的數值積分。理解「樣本內夏普越高，樣本外失望越大」的數學直覺，重構因子測試 pipeline。
