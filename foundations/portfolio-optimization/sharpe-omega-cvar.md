**X-Ray.** 本框架將組合構建從「靜態求解器」遷移至「動態梯度流」，在 Pareto 前沿上犧牲了部分尾部防禦（MDD 劣於 NCO），換取了對非線性時間依賴與交易成本的內生感知。它解決了 MVO 對預測誤差非線性放大的工程坑，但並未打開高維橫截面選股的 envelope。對量化讀者而言，其價值不在於替代因子挖掘，而在於提供了一套「將實盤約束（Long-only、成本、風險平價）編碼為可微損失」的範式。95% LCB 調參邏輯是對 Walk-Forward 過擬合的實戰級修正，但協方差收縮的反向傳播穩定性與 Softmax 的集中度缺陷，仍是實盤落地的關鍵瓶頸。該方法適合中低頻行業輪動或因子組合權重分配，而非全市場選股。

## §1 · 架構 / Core Mechanism
| 改動維度 | 傳統二階段流程 | 本框架 |
|---|---|---|
| 輸出層 | 回歸預測超額收益率 | Softmax 直接輸出 N 維權重向量 |
| 目標函數 | MSE / IC 最大化 | 可微複合損失 (Sharpe/Omega/CVaR/RP) |
| 超參搜索 | 驗證集平均收益最大化 | 95% LCB 信息比率極大極小優化 |

⚡ **Eureka:** 將不可微的組合績效指標透過 Softplus 與變分公式轉為可回傳梯度的損失項，讓網路直接「學習」配置而非預測。

```
[T天, 251維特徵] -> [LSTM/Attention Encoder] -> [Softmax] -> [W (N維權重)]
                                                      |
                                                      v
[Loss = α·L_SmoothSharpe + β·L_SmoothOmega + γ·L_CVaR + δ·L_RiskParity] -> [Backprop]
```

## §2 · 數學層
📌 **Napkin Formula:**
`L_SmoothSharpe = -log(Softplus(μ/σ + ε))`
`L_CVaR = (VaR + E[max(0, r - VaR)]/α) / σ` (VaR 估計使用 Stop-gradient)
`L_RP = Σ(risk_contribution_i - avg_risk_contribution)^2 / N`
**複雜度:** O(N²) 協方差估計與反傳。
**直覺:** 用平滑算子替換硬閾值與不可導運算，使組合指標成為網路可優化的標的。
**Loss/訓練細節:** 線性收縮協方差矩陣，梯度裁剪 (clip_grad_norm=0.5)，95% LCB 驅動 Optuna 搜索。

**解讀:** Δ 的來源可拆解為兩部分：收益與夏普的顯著提升（經 Bonferroni 矯正單尾 t 檢驗，vs S&P 500 與 vs NCO 均達統計顯著，導讀未披露具體 p 值）屬真實 capability，源於損失函數直接對齊風險調整收益且已扣除首日買賣價差成本。然而 MDD Δ 顯示本方法在尾部防禦上主動讓步於 NCO，反映 Sharpe/Omega 主導損失對極端回撤的懲罰力度不足。此外，測試期涵蓋暴力加息週期，模型在熊市初期存在 60-120 天適應滯後，部分 Δ 可能隱含了對特定宏觀環境的週期性暴露，而非純粹的 alpha 穩定性。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations:** 熊市初期適應滯後（1-2 個季度）、協方差收縮反傳易引發梯度爆炸、僅限 50 只資產規模。
**6.2 推斷的隱含假設:** 滾動窗口內市場結構相對平穩、買賣價差足以代表真實交易成本、Long-only 約束可接受、線性收縮協方差矩陣的反傳數學上穩定且數值可解。
