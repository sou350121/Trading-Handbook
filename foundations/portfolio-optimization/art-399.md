# 异质性信念均衡模型 解構

> **發布**：2026-06-20 · （無 venue）

> **核心定位**：落點於「風險擇時」與「中長週期」軸，填補了理性定價模型與行為外推調查之間的數據斷層。透過重構上世紀五十年代中期至本世紀二十年代中期的機構預期，將宏觀估值信號錨定至微觀因子模型的非平穩性痛點轉化為可驗證的信念分歧因子。

**TL;DR:** ① 重構近七十年 Value Line 機構預期序列，驗證異質性信念均衡模型。② 核心 trick 為 AWS Textract OCR 数字化 + Stambaugh 偏誤修正 + Campbell-Shiller 分解，量化信念分歧對量價的決定性影響。③ 對「風險擇時」軸的關鍵價值在於提供逆週期、抗外推偏差的宏觀配置信號。④ 導讀給出四年期預測解釋力達百分之十四左右，六年期累積至近四分之一。

**X-Ray.** 本方法在五軸 Pareto 前沿上明確放棄了短週期高頻 Alpha 的爭奪，將數據工程傾斜至「數據可得性」與「中長週期」的交叉點。它解決了量化實證中兩大工程坑：一是傳統調查數據樣本過短導致的 Stambaugh 偏誤，二是宏觀估值與微觀因子錨定時的非平穩性。其預測 Envelope 無法打開短週期交易信號或純量價動能因子，因為模型本質是基於會計恒等式與折現模型的結構性回歸，而非端到端黑盒。對量化讀者的意義不在於直接套用作為交易信號，而在於提供一個經過長週期驗證的「信念分歧」濾波器。在組合配置層面，該序列可作為波動率調節與倉位逆轉的宏觀閾值；在因子研究層面，它揭示了估值波動近六成由預期收益率變動驅動，這直接挑戰了傳統基本面因子對盈利增長的過度依賴。實盤落地需解決數據發布滯後與覆盤修正問題，且其預測力建立在機構與散戶持續存在信念分歧的市場結構上，若市場機構化程度過高導致分歧收斂，信號衰減將是必然。

## §1 · 架構 / Core Mechanism
| 維度 | 傳統學術/行為金融基線 | 本方法改動 |
|---|---|---|
| 數據源與長度 | 調查問卷/CFO數據（樣本通常僅十餘年） | OCR重構Value Line歷史檔案（近七十年） |
| 預期度量方式 | 模糊趨勢問卷或隱含資本成本代理 | 基於DDM變體的複利方程數值求解（具體財務預測） |
| 統計偏誤處理 | 忽略長期預測回歸的序列相關性 | 引入Stambaugh偏誤修正（平穩塊自助法+解析糾正） |

⚡ **Eureka:** 用會計恒等式與折現方程將「主觀預期」轉化為可數值求解的隱含收益率，而非依賴問卷打分。
```
[VL紙質合訂本/縮微膠卷] -> (AWS Textract OCR) -> [財務預測表格重建]
      |
      v
[當前股價/盈利預測/目標價] -> (DDM變體複利方程) -> [隱含主觀年化預期收益率]
      |
      v
[散戶外推預期 vs 機構逆週期預期] -> (信念分歧絕對值) -> [交易量/波動率預測]
```

## §2 · 數學層
📌 **Napkin Formula:**
`P_t = D_{t+1}/(1+r) + ... + P_{t+4}/(1+r)^4` （高次代數方程數值求解 r）
`ΔCAPE ≈ ΔE[r] (近六成) + ΔE[Re-rating] (近四成) + ΔE[CashFlow] (百分之三左右)`
**直覺:** 預期收益率 r 通過非線性折現方程隱式定義；Campbell-Shiller 分解將市盈率變動拆解為貼現率預期、估值重估與基本面增長，驗證機構預期主導定價。
**Loss/訓練:** 無梯度下降 Loss。依賴統計回歸與自助法偏差修正，計算複雜度為線性時間序列運算。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations:** 導讀未明確列出作者自述的 limitations 段落，僅提及需應對 Stambaugh 偏誤。
**6.2 推斷的隱含假設:**
- **Regime 依賴:** 依賴機構與散戶持續存在信念分歧的市場結構。若市場完全機構化或量化資金同質化，分歧指標將失效。
- **容量/成本:** 中長週期配置信號，不適合高頻交易。數據發布存在滯後，實盤需處理覆盤修正（look-ahead bias）。
- **數據泄漏:** OCR 與模糊匹配 PERMNO 過程若未嚴格按時間戳切割，易引入未來信息。
- **測量誤差:** DDM 變體依賴固定股息支付率假設，若公司回購/分紅政策結構性改變，隱含收益率求解將產生偏差。

## §8 · For the Reader
- **因子研究員:** 將信念分歧絕對值作為交互項加入傳統估值因子，檢驗其在高波動 regime 下的非線性暴露。避免直接做多/做空預期收益率序列。
- **組合配置/宏觀策略:** 利用六年期預測累積效應（近四分之一）設定戰略資產配置（SAA）的逆週期閾值。需對沖數據發布滯後風險。
- **高頻/執行研究:** 本方法不適用。但信念分歧指標可作為日內波動率突升的先行宏觀狀態變量，用於動態調整執行算法的風險參數。
- **LLM-agent/研究學生:** 關注 OCR 重構與財務勾稽驗證的工程細節。該數據管道展示了如何將非結構化歷史文檔轉化為結構化時間序列，是量化數據工程的典範。
