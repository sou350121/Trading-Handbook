# SAMBA 解構

> **發布**：2025-02-14 · （無 venue）
> **QuantML 導讀**：[SAMBA：基于Mamba架构与图神经网络的股票价格预测模型](https://mp.weixin.qq.com/s?__biz=Mzg2MzAwNzM0NQ==&mid=2247489266&idx=1&sn=321d2a1506b2e8e1c867d95c7cf1caf4&chksm=ce7e71ecf909f8fa2932c9ed166e98b841595850fb6bf3256b6e29b953acf4029c6362b0c2e3#rd)
> **核心定位**：落點於「日频波段 × 監督回歸 × 端到端表徵」軸，以雙向 Mamba 線性複雜度替代 Transformer 二次瓶頸，並透過自適應圖卷積動態建模特徵交互，解決長序列依賴與跨股票關聯算力的 prior gap。

|:-:|:-:|:-:|:-:|:-:|

**Status:** v0.5 — 基於 QuantML 導讀 + 原論文（如有）。benchmark 細節待升 v1。
**TL;DR:** ① 以雙向 Mamba 塊處理長序列時間依賴，結合自適應圖卷積塊建模每日特徵交互，輸出 1 日收益率回歸預測。② 核心 trick 為高斯核動態建圖 + 矩陣分解降參 + 雙向掃描過濾無關數據。③ 對「全自動黑盒」軸★，將計算複雜度壓至線性級別，使長窗口日頻預測具備實盤推理可行性。④ 導讀未給量化結果。

**X-Ray.** SAMBA 的本質是「序列狀態空間模型 × 動態圖神經網絡」的並行解耦。它避開了 Transformer 在日頻長窗口下的 O(N²) 顯存爆炸，同時用 AGC 的任務特定高斯核取代靜態鄰接矩陣，讓特徵交互從先驗硬編碼轉向數據驅動軟學習。對量化實戰而言，這意味著回測窗可拉長而不觸發 OOM，且能自動捕捉風格輪動下的隱性共變結構。但該架構未處理非平穩性與結構性斷點，雙向掃描在實盤推理時存在未來信息泄漏風險，且回歸目標直接對齊收益率，缺乏對尾部風險的顯式約束，在 regime shift 時易產生過度自信預測。

## §1 · 架構 / Core Mechanism
| 改動維度 | 前作/基線 | SAMBA 改動 | 工程意義 |
|---|---|---|---|
| 序列建模 | Transformer (O(N²)) / MambaStock (單向) | 雙向 Mamba (BI-Mamba) + 殘差歸一化 + FFN | 線性複雜度捕捉長程依賴，雙向彌補單向信息損耗 |
| 特徵交互 | 靜態圖 (行業/市值) / MLP 獨立處理 | 自適應圖卷積 (AGC) + 高斯核動態建圖 + 矩陣分解 | 端到端學習特徵依賴，矩陣分解壓參數防過擬合 |
| 目標函數 | 絕對價格預測 / 分類 | 1 日收益率回歸 (Regression) | 對齊交易實際 PnL 驅動，避免價格非平穩性干擾 |

⚡ **Eureka:** 用可學習縮放因子控制高斯核衰減，讓圖結構隨市場狀態動態重構，而非依賴靜態先驗。
```
Input [T, F] → BI-Mamba (Fwd/Bwd) → ResNorm → FFN → ResNorm → Y [T, F]
                                                      ↓
                                              AGC (Gaussian Kernel + Chebyshev K)
                                                      ↓
                                              Linear Projection → Return [T, 1]
```

## §2 · 數學層
📌 **Napkin Formula:**
$h_t = \bar{A} h_{t-1} + \bar{B} x_t$ (Mamba SSM 離散化狀態更新，複雜度 $O(T \cdot D)$)
$Z = \sum_{k=0}^{K} \theta_k T_k(\tilde{L}) X$ (AGC 切比雪夫多項式圖卷積，$\tilde{L}$ 為動態拉普拉斯矩陣)
**直覺:** Mamba 透過選擇性掃描將無關時間步的 $\bar{B}$ 壓縮至近零，實現類似注意力但線性開銷的過濾；AGC 將特徵視為節點，利用矩陣分解將權重參數化，使圖卷積在 $K=3$ 階近似下保持輕量。Loss 與訓練細節導讀未披露。

**解讀:** Δ 均為相對「第二好的模型」的提升百分比，非絕對數值。IC/RIC 提升幅度極大（如 NASDAQ IC 提高 85.38%），暗示基線模型在該數據集上表現極差或存在數據泄漏/前瞻偏差（靜態 80/5/15 分割未處理非平穩性，易導致訓練集分佈覆蓋測試集）。RMSE 提升較溫和（6.90%-11.72%），符合回歸任務對異常值不敏感的特性。計算複雜度排名第五，雖優於 Transformer/AGCRN，但高於 MLP 基線，實盤推理延遲需實測。未計交易成本與滑點，純預測指標無法直接映射至 Sharpe/IR。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations:** 未明確列出。僅提及市場行為受多因素影響且具非線性，傳統模型有局限。
**6.2 推斷隱含假設:**
- **Regime 依賴:** 雙向 Mamba 在訓練時利用雙向信息，實盤推理必須嚴格因果截斷（僅用歷史），否則產生嚴重前瞻偏差。
- **數據泄漏風險:** 靜態時間切分（80/5/15）未處理宏觀週期切換，測試集可能包含訓練集已學習的結構性模式。
- **容量/成本:** 日頻波段策略容量取決於股票池規模與流動性，導讀未披露交易成本假設，高 RIC 未必轉化為實盤 Alpha。
- **圖結構穩定性:** 高斯核動態建圖依賴特徵分佈穩定性，在極端波動或流動性枯竭時，可學習縮放因子可能發散或收斂至無效圖。

- ✅ 正確答：「SAMBA 用雙向 Mamba 解決長序列線性建模，AGC 用高斯核動態學習特徵依賴。實盤需嚴格因果截斷雙向狀態，且靜態數據劃分需改為滾動窗口以防非平穩性泄漏。」
- ❌ 錯答：「雙向 Mamba 可以直接用於實盤預測，因為它已經過濾了無關數據。」（忽略因果性與前瞻偏差）

**7.1 可證偽預測:** 若將靜態 80/5/15 劃分改為嚴格滾動窗口（Walk-Forward）並加入 5bp 交易成本，SAMBA 的 IC 提升幅度將收斂至 10%-20% 區間，且 RMSE 優勢在波動率 regime 切換時失效。（預測日期：2025-12-31 前驗證）

## §8 · For the Reader
- **因子研究員:** 將 AGC 的高斯核動態建圖思想移植至因子正交化流程，替代靜態行業/市值分組，觀察特徵共變結構的時變性。
- **高頻/日頻執行:** 關注 Mamba 線性推理延遲，在不同回看窗口長度下實測單步推斷耗時（導讀未披露具體窗口設定），評估是否滿足日頻盤前批量計算 SLA。
- **組合配置:** 回歸目標直接輸出收益率，需疊加風險模型（如 Barra）或 CVaR 約束，避免模型在尾部風險區間過度加倉。
- **LLM-Agent/RL 策略:** 將 SAMBA 的預測分數作為 RL 狀態空間的觀測輸入，而非直接作為動作，利用 RL 處理非凸交易成本與執行約束。
- **研究學生:** 復現時重點驗證矩陣分解對 AGC 參數量的壓縮效果，對比 K=1/3/5 階切比雪夫近似在 IC 上的邊際收益。
