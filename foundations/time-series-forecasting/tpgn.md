**Status:** v0.5 — 基於 QuantML 導讀 + 原論文（如有）。benchmark 細節待升 v1。
**TL;DR:** ① 提出 PGN 作為 RNN 繼任者，並擴展為 TPGN 框架處理長程時序預測。② 核心 trick 為一維序列二維重塑 + 雙分支並行提取長短期特徵 + 單門控全局融合，將信息傳播路徑縮短至 O(1)。③ 對「端到端表徵」軸★ 意義在於以並行計算與單門控機制徹底繞過 RNN 的梯度消失/順序計算瓶頸，同時保留語義捕捉能力。④ 導讀給出平均 MSE 降低 12.35%、MAE 降低 7.25%，且為單層架構。

**X-Ray.** 放回五軸 Pareto，TPGN 在「中長週期」與「監督回歸」軸上，用 O(1) 信息路徑與 O(VL) 複雜度取代了 Transformer 的二次方瓶頸與 RNN 的順序延遲。它解了舊工程坑：梯度爆炸/消失、順序計算延遲、多層堆疊帶來的記憶體開銷。預測它打不開的 envelope：缺乏對變量間異質性與圖結構關係的顯式建模，在跨資產聯動或高頻微結構跳躍場景下可能失效。對量化讀者意義：提供極輕量、易並行部署的時序特徵提取器，適合中頻因子合成與低延遲推理管道，但需警惕其在非週期性、結構突變 regime 下的泛化邊界。

## §1 · 架構 / Core Mechanism
**1.1 三大改動 vs 前作**
| 改動維度 | 前作 (RNN/Transformer) | TPGN / PGN | 工程意義 |
|---|---|---|---|
| 信息傳播路徑 | O(L) 順序遞歸 / O(1) 注意力 | O(1) 並行聚合 | 繞過梯度消失與計算瓶頸 |
| 序列建模維度 | 一維遞歸 / 逐點注意力 | 一維重塑二維 + 雙分支 | 解耦長短期週期語義 |
| 門控機制 | 多門控 (GRU/LSTM) 或無 | 單門控全局控制 | 降低參數開銷與訓練難度 |

**1.2 ⚡ Eureka 一句話 trick**
將一維序列重塑為二維（行=短期/列=長期），用單個門控同時控制所有時間步的信息選擇與融合，實現 O(1) 路徑與並行計算。

**1.3 信息流 ASCII 圖**
```
Input (1D, L) -> [Reshape 2D (P x R)] 
                -> [Long Branch: PGN along R] 
                -> [Short Branch: Linear -> Patch -> Global] 
                -> [Concat] -> [Single Gate Fusion] -> [Linear Prediction]
```

## §2 · 數學層
**📌 Napkin Formula**
PGN: $H = \sigma(W_h X + b_h) \odot \tanh(W_o X + b_o)$，複雜度 O(L)。
TPGN: $X \in \mathbb{R}^{L \times V} \to \mathbb{R}^{P \times R}$，Long: $H_{long} = PGN(X_{long})$，Short: $H_{short} = Linear(Patch(X_{short}))$，Output: $Y = Linear([H_{long}; H_{short}])$，複雜度 O(VL)。
**直覺**：線性層並行聚合歷史，單門控做元素積篩選，避免遞歸狀態傳遞與多層堆疊。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations**
變量維度建模受限（融合易引入雜訊，獨立處理無法捕捉變量關係）；週期性特徵捕捉可能不徹底；未討論結構突變或極端行情下的魯棒性。
**6.2 推斷的隱含假設**
強依賴數據的週期性與平穩性（中長週期假設）；容量假設未披露，但單層線性架構暗示參數容量有限，可能不適合高維稀疏因子；數據泄漏風險低（公開基準集），但實盤需警惕 look-ahead bias 在重塑二維過程中的邊界處理。

## §7 · 對比 & 面試 Tip
| 同軸對手 | 關鍵差異軸 | Open? | Status |
|---|---|---|---|
| RNN/LSTM | 順序遞歸 vs 並行 O(1) 路徑 | 開源 | 被取代 |
| Transformer | O(L^2) 注意力 vs O(VL) 線性/並行 | 開源 | 互補/替代 |
| TimesNet | 二維 CNN vs PGN 單門控 | 開源 | 競爭 |

🎤 **Interview Tip**
- **正確答**：TPGN 用單門控+二維重塑將信息路徑壓至 O(1)，繞過 RNN 梯度瓶頸，適合中頻週期因子提取。
- **錯答**：認為它能替代 Transformer 處理所有高維異質數據或具備自回歸推理能力。

**7.1 可證偽預測帶日期**
若 2025-Q3 前無開源實現在公開量化數據集（如 A 股日頻/小時頻）上復現 MSE/MAE 降幅，則其泛化邊界受限於氣象/能源數據的強週期性假設。

## §8 · For the Reader
- **因子研究員**：直接替換 LSTM 作為時序特徵提取器，注意二維重塑的視窗對齊與週期假設。
- **高頻執行**：不適用，架構針對中長週期設計，未優化微結構跳躍與低延遲推理。
- **組合配置**：可作為低延遲預處理器，但需驗證跨資產聯動捕捉能力與容量上限。
- **LLM-agent/RL 策略**：可嵌入狀態編碼器，但單門控可能限制複雜策略的表徵容量。
- **研究學生**：重點複現 PGN 的 O(1) 路徑與單門控機制，對比 GRU 的參數效率與梯度流。
