**X-Ray.** 放回五軸 Pareto，ReFocus 在「中長週期×監督回歸」軸上，用頻域概率分佈替代了傳統時域注意力或固定濾波器，直接對齊了量價數據中常見的「低頻趨勢主導、中頻結構缺失」特徵。它解了 RevIN 與高低通濾波器無法處理中頻能量匱乏的工程坑，但 EKPB 的跨通道 Softmax 依賴強假設變量間存在頻譜共鳴，這在金融市場（異質性高、regime 切換快）可能面臨容量瓶頸。預測其打不開的 envelope 是：高頻微結構或極端波動下的非平穩跳躍，因為 FFT 本身對非平穩性敏感，而 AMEO 僅做能量重分配而非動態 regime 切換。對量化讀者意義：提供了一條「頻域特徵工程自動化」的路徑，可作為多因子模型的前置頻譜濾波層，但需警惕跨通道共享假設在個股異質性上的失效。

## §1 · 架構 / Core Mechanism
| 改動模塊 | vs 前作（iTransformer/RevIN/濾波器） | 工程意圖 |
|---|---|---|
| AMEO | 取代固定高通/低通濾波與 RevIN 歸一化 | 基於卷積與殘差學習，為低頻分配更高縮放因子，強制分散能量填補中頻隙 |
| EKPB | 取代純時域 Cross-Attention 或獨立通道建模 | 計算頻譜能量平方 → 跨通道 Softmax → 概率採樣共享關鍵頻，參數更少 |
| KET | 取代單一分佈訓練 | 隨機混入其他通道頻譜合成偽樣本，真實/合成數據交替訓練防過平滑 |

⚡ **Eureka 一句話 trick**：頻域不是「濾波」而是「重分配概率」——用 Softmax 把能量轉成跨通道選擇權，讓模型自己決定哪些頻率該被跨變量共享。

**信息流 ASCII:**
```
Input X → [AMEO: 能量重分配] → FFT → [EKPB: MLP→FFT→Energy→Softmax→Pick→iFFT]
       → Add&Norm → MLP → Output Y
       ↖ KET: 交替訓練 (Real ↔ Synthetic Mix-up)
```

## §2 · 數學層
📌 **Napkin Formula:**
$E_c = |FFT(X_c)|^2$
$P_c = \text{Softmax}(E_c)$
$X_{shared} = \text{iFFT}(\sum_{c} P_c \cdot FFT(X_c))$
複雜度：FFT $O(N \log N)$ / 通道；跨通道 Softmax $O(C)$。

**直覺**：能量平方映射為概率分佈，跨通道加權平均提取共性，殘差連接保留通道個性。
**Loss/訓練**：MSE Loss。交替在真實數據與 KET 合成數據上訓練，保留真實樣本頻譜依賴性，緩解純合成數據導致的過度平滑與訓練不穩定。

## §5 · 評測 / Benchmark
| 數據集/市場 | Metric | 前SOTA | 本方法 | Δ |
|---|---|---|---|---|
| Traffic | MSE | 未披露 | 未披露 | -4% |
| ECL | MSE | 未披露 | 未披露 | -6% |
| Solar | MSE | 未披露 | 未披露 | -5% |

**解讀**：Δ 為相對 iTransformer 的 MSE 降幅，屬預測誤差指標，非金融收益指標。4-6% 的 MSE 改善在學術基準屬顯著，但金融實盤需轉換為 IR/Sharpe 或交易成本敏感度。EKPB 單模塊控制變量測試顯示 MSE 從 0.171 降至 0.145（Δ -15%），此為純模塊貢獻，非整體模型表現。需警惕學術數據集（交通/電力/太陽能）的低頻平穩性與金融高頻非平穩性的分佈偏移；Δ 中的真 capability 來自頻譜能量重分配與概率共享，但潛在過擬合風險在於跨通道 Softmax 可能將學術數據中的偽相關頻率過度放大，且導讀未計入任何交易成本或前瞻偏差。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations**：依賴 FFT 假設週期性/平穩性；跨通道共享頻假設在異質性強的數據可能失效；KET 交替訓練增加訓練複雜度。
**6.2 推斷的隱含假設**：
- **Regime 依賴**：頻譜結構在樣本外需保持相對穩定，否則 AMEO 的重分配因子將滯後。
- **容量**：跨通道 Softmax 隨通道數 $C$ 增長，極高維度下可能稀釋關鍵頻信號。
- **成本/數據泄漏/Survivorship**：學術基準無交易成本；標準劃分降低泄漏風險；但金融實盤需處理停牌/退市樣本的 Survivorship Bias，導讀未提及。

## §8 · For the Reader
- **因子研究員**：將 AMEO 視為自動化頻譜濾波層，可替換傳統 HP 濾波或帶通濾波，提取中頻結構因子，但需對金融數據做差分/對數收益率處理以滿足 FFT 平穩性前提。
- **組合配置**：EKPB 的跨通道 Softmax 可類比為動態權重分配，但需警惕金融資產異質性導致的「偽共享頻」；建議在行業/風格組內使用，而非全市場跨資產。
- **研究學生**：複現時重點驗證 KET 交替訓練對過擬合的抑制效果，並測試在非平穩金融數據上 FFT 的邊界條件；可嘗試將 EKPB 概率分佈轉化為可解釋的頻譜因子暴露矩陣。
