# HAR-VIX 解構（HAR-VIX）

> **發布**：2024-07-11 · （無 venue）
> **QuantML 導讀**：[线性模型足矣？传统HAR模型与机器学习在实现波动率预测中的对比](https://mp.weixin.qq.com/s?__biz=Mzg2MzAwNzM0NQ==&mid=2247485388&idx=1&sn=80428bd4ba936be537bc300ab55ef7d5&chksm=ce7e60d2f909e9c46231f77be22b1f6fdb69a5d1d1c0f607ba740b5572eba0f09e79f5fccfbd#rd)
> **核心定位**：落點於量價表格與日頻波段的監督回歸框架，以風險擇時為 Alpha 生成機制。解了「過度追求 ML 非線性卻忽略估計策略紀律」的 prior gap，證明線性異質自回歸在正確工程配置下具備實戰級別的可解釋與經濟效用。

**Status:** v0.5 — 基於 QuantML 導讀 + 原論文（如有）。benchmark 細節待升 v1。
**TL;DR:** 系統性對比 HAR 與主流 ML 模型在實現波動率（RV）預測上的表現。核心 trick 在於嚴格校準滾動窗口步長與重估頻率，並引入 VIX 作為全局狀態因子。這直接強化了風險擇時軸的落地性：線性模型能以極低計算成本匹敵或超越黑盒 ML。導讀未給量化結果，僅以 MCS 與經濟效用框架定性驗證其優勢。

**X-Ray.** 在量價表格與日頻波段的 Pareto 前沿上，HAR-VIX 切中了量化實戰中「模型複雜度與過擬合」的經典矛盾。多數團隊盲目堆疊 RF/FFNN 卻忽略估計策略（重估頻率/窗口步長），導致因子在樣本外迅速衰減。本文將 HAR 的線性結構與 VIX 的全局狀態信息耦合，證明在正確的工程配置下，輕量級監督回歸足以捕獲波動率聚集與異質性特徵。它打不開的 envelope 在於無法捕捉高頻微結構跳躍或極端 regime 下的非線性斷裂，但這恰恰是日頻波段策略的合理邊界。對量化讀者而言，其意義不在於提出新架構，而在於提供了一套可證偽的「估計策略校準」範式：當 ML 的超參數調優成本超過其邊際預測增益時，回歸基礎計量經濟學的工程紀律才是 Alpha 可持續的關鍵。

## §1 · 架構 / Core Mechanism
**1.1 三大改動 vs 前作**
| 改動維度 | 前作/基線 (HAR-Corsi / 純 ML) | HAR-VIX 本框架 | 工程意義 |
|---|---|---|---|
| 因子結構 | 僅依賴歷史 RV (日/周/月) | 引入 VIX 作為全局宏觀狀態因子 | 補齊跨市場風險溢價，降低純歷史序列的滯後性 |
| 估計策略 | 固定窗口或靜態訓練 | 嚴格滾動窗口 + 動態重估頻率校準 | 解決樣本外衰減，匹配日頻波段的 regime 切換節奏 |
| 驗證框架 | 單一統計損失 (MSE/MAE) | MCS 程序 + 實現效用 (Realized Utility) 經濟評估 | 從「預測準確」轉向「交易可執行」，過濾過擬合信號 |

**1.2 ⚡ Eureka**
滾動步長與重估頻率決定線性模型的樣本外壽命，VIX 注入全局狀態信息即可讓輕量級回歸在經濟效用上壓倒黑盒 ML。

**1.3 信息流 ASCII**
```
Intraday TAQ (5-min) → Log RV Calculation
       ↓
[Rolling Window Estimator] ← (Step/Freq Calibration)
       ↓
HAR-VIX (Linear)  vs  Lasso/RF/GBT/FFNN (ML)
       ↓
MSE / QLIKE  →  MCS (95% Conf)  →  Realized Utility (Cost-Adjusted)
       ↓
Risk-Timing Signal (日频波段)
```

## §2 · 數學層
📌 **Napkin Formula**
`RV_t = β0 + β1·RV_{t-1}^{(d)} + β2·RV_{t-1}^{(w)} + β3·RV_{t-1}^{(m)} + β4·VIX_{t-1} + ε_t`
**複雜度**：O(N) 每步滾動更新，矩陣求逆或 WLS 閉式解，無迭代收斂風險。
**直覺**：異質時間尺度 (日/周/月) 加權捕獲波動率記憶性與長程相關性；VIX 項將隱含波動率溢價映射為狀態變量，修正純歷史 RV 的順週期偏差。
**Loss/訓練**：統計層使用 MSE 與 QLIKE；經濟層使用實現效用函數（已扣除交易成本代理）；估計採用滾動窗口 OLS/WLS，重估頻率為超參。

## §3 · 數據層
- **規模/頻率/市場**：初始 6,061 支股票，過濾後 1,445 支保持一致。NYSE TAQ 數據，5分鐘對數回報率計算日度 RV。
- **時段**：2015-01-01 至 2023-10-31。
- **樣本外與容量假設**：嚴格滾動窗口劃分 train/test，無前瞻性泄漏。日頻波段信號容量極高（1,445 支流動性標的），適合中大型資金分散配置。子樣本 (27 支 DJIA / 69 支 Nasdaq-100) 驗證結果一致。

**解讀**：Δ 的真實性不在於統計損失的微小壓制，而在於 **估計策略紀律** 與 **成本調整後的經濟效用**。ML 模型在 MSE 上可能短暫領先，但缺乏重估頻率校準會導致樣本外信號衰減；HAR-VIX 的優勢來自於將預測準確性直接映射為可執行的組合效用（使用過去 9 個月買賣價差中位數作為成本代理）。若實盤中滑點超過該代理閾值，經濟效用優勢將迅速收斂。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations**
線性結構無法捕獲極端行情下的非線性斷裂；依賴 5分鐘 RV 估計器，可能受微結構噪聲干擾；經濟效用評估使用固定成本代理，未動態適配市場流動性枯竭期。

**6.2 推斷的隱含假設**
- **Regime 依賴**：VIX 與 RV 的線性耦合關係在常態市場有效，但面臨流動性危機或波動率跳躍時可能失效。
- **容量/成本**：日頻波段適合中頻調倉；若實盤交易成本顯著高於過去 9 個月買賣價差中位數，經濟效用將轉負。
- **數據泄漏**：滾動窗口必須嚴格 purged，否則重估頻率校準會產生虛假樣本外優勢。
- **Survivorship**：1,445 支股票過濾了退市/流動性過低標的，實盤需處理存續偏差。

## §7 · 對比 & 面試 Tip
| 同軸對手 | 關鍵差異軸 | Open? | Status |
|---|---|---|---|
| HAR-Corsi (2009) | 無全局狀態因子 / 靜態估計 | 開源 | 經典基線 |
| RF / GBT / FFNN | 非線性捕獲 / 黑盒 / 高調參成本 | 開源 | 實戰常過擬合 |
| HAR-VIX | VIX 狀態注入 / 滾動步長校準 / 經濟效用驗證 | TBD | v0.5 |

🎤 **Interview Tip**
- **正確答**：HAR-VIX 的優勢不在模型複雜度，而在估計策略的工程紀律（重估頻率/窗口步長）與全局狀態因子 (VIX) 的耦合。線性模型在正確配置下能以極低成本實現與 ML 相當的預測，且 MCS 與經濟效用驗證確保了信號的可執行性。
- **錯答**：因為 HAR 是線性模型所以不會過擬合，或者認為 ML 永遠優於傳統計量模型。忽略重估頻率與成本調整的討論。

**7.1 可證偽預測**
若 2025-Q2 波動率 regime 切換至非線性跳躍主導（如 VIX 與 RV 相關性斷裂），HAR-VIX 的 MCS 排名將滑落至非線性 ML 模型之後，且實現效用轉負。

## §8 · For the Reader
- **因子研究員**：將重估頻率與滾動步長納入因子校驗流水線，替代固定窗口回測；優先使用 QLIKE 而非 MSE 評估波動率因子。
- **組合配置**：利用 HAR-VIX 輸出作為波動率預算的動態約束；在 MCS 置信集內做模型平均，而非單一擇優。
- **高頻執行/系統開發**：日頻波段信號對滑點敏感，實盤需將成本代理替換為動態盤口深度模型；HAR 的 O(N) 複雜度適合低延遲預計算。
- **LLM-agent / RL 策略**：將 HAR-VIX 的輸出作為狀態空間的觀測變量 (Observation)，而非直接作為 Action；RL 代理可學習何時切換至非線性模型。
- **研究學生**：複現時嚴格實現 purged rolling window；對比 WLS 與 OLS 在異方差 RV 序列上的 MCS 差異，理解計量假設對實戰的影響。
