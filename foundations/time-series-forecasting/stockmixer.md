> **核心定位**：落點於「端到端表征 × 全自动黑盒」軸，以純 MLP 架構替代 RNN/GNN 處理多變量時間序列。解了圖結構先驗依賴與遞歸梯度瓶頸的 prior gap，將截面交互降維為可並行的超圖式線性操作。

|:-:|:-:|:-:|:-:|:-:|

**Status:** v0.5 — 基於 QuantML 導讀 + 原論文（如有）。benchmark 細節待升 v1。
**TL;DR:** ① 提出純 MLP 架構做日频收益預測，棄用遞歸與圖消息傳遞。② 核心 trick 是時間混合塊的上三角因果掩碼結合多尺度 patch，以及股票混合塊的「個股↔市場」超圖式解耦。③ 對「端到端表征」軸★：以極低參數量與計算開銷實現跨截面信息交換，訓練穩定性顯著提升。④ 導讀明確指出其在多數指標上平均相對性能提升 7.6%，10.8% 和 10.9%（p < 0.01）。

**X-Ray.** 將 StockMixer 放回五軸 Pareto，它本質是「輕量級 MLP 對沖高頻混沌」的極端嘗試。它解了兩個舊工程坑：一是 RNN 梯度消失與 GNN 鄰接矩陣稀疏/先驗依賴導致的訓練不穩；二是全連接交互在樣本有限時的嚴重過擬合。其時間掩碼與超圖解耦直接對應量化實盤中的「因果防洩漏」與「截面因子稀疏性」。但預測它打不開的 envelope 在於：純 MLP 缺乏對非線性 regime shift 的動態路由能力，當市場波動率結構發生斷裂，固定權重混合塊會迅速失效。對量化讀者的意義是提供了一個極易並行化、可插拔的截面表征底座，但必須搭配嚴格的波動率過濾與動態權重衰減，否則回測優異僅是歷史樣本內的線性拟合。

## §1 · 架構 / Core Mechanism
| 改動維度 | 前作 (MLP-Mixer / RNN / GNN) | StockMixer 設計 | 工程意圖 |
|:---|:---|:---|:---|
| **指標混合** | 空間/通道平等混合 | 轉置時間/指標維度做特徵通道混合 | 解耦單標的內部因子共線性 |
| **時間混合** | 全連接平等交互 / 遞歸狀態 | 上三角掩碼全連接 + 多尺度 (k∈{1, 2, 4}) 池化 | 強制因果序防未來洩漏，緩解短序列時間偏差 |
| **股票混合** | N×N 全連接圖消息傳遞 | 隱藏維度設為超參數 m，解耦為 Stock↔Market | 壓縮弱相關噪音，以超圖式降維/升維抗過擬合 |

⚡ **Eureka:** 用矩陣上三角切斷未來信息，用超圖式降維壓縮截面交互，以 MLP 實現「防洩漏+抗過擬合」。
```text
輸入 X(N,T,F) 
  → 指標混合 (轉置+FC) 
  → 時間混合 (上三角掩碼 + 多尺度k∈{1,2,4}池化) 
  → H(N,d) 
  → 股票混合 (N→m→N 超圖解耦) 
  → 拼接+FC降維 
  → 預測 r(N,1)
```

## §2 · 數學層
📌 **Napkin Formula:**
`y = σ(W2 * σ(W1 * x))` （標準混合塊）
`L = MSE(r̂, r) + α * RankingLoss(r̂_rank, r_rank)` （α=0.1）
**複雜度:** O(T*F*h + T^2 + N*m) 遠低於 GNN 的 O(E) 與 RNN 的 O(T*hidden^2)。
**直覺:** 掩碼矩陣強制時間因果序，超圖壓縮消除弱相關截面噪音。訓練使用 Adam，學習率 1e−3，每個實驗重複 3 次報告平均性能。激活函數實證顯示 ReLU 與 HardSwish 優於 GELU。

## §3 · 數據層
- **資料規模/頻率/市場/時段:** 美股日频。回顧窗口 T=16，預測次日收盤價換算 1 日回報率。市場涵蓋 NASDAQ、NYSE、S&P500。NASDAQ/NYSE 時段為 2013年2月1日至2017年12月8日。
- **怎麼來:** 過濾異常模式與小盤股，保持代表性屬性。S&P500 含行業信息。圖/超圖結構依原始論文預處理構建。
- **樣本外與容量假設:** 樣本通過交易日滑動 16 天窗口生成。未提及嚴格 Walk-Forward 或時間序列交叉驗證細節，僅報告平均性能。假設短期截面相關性具持續性，且近期信息主導定價。

## §5 · 評測 / Benchmark
| 數據集/市場 | Metric | 前SOTA | 本方法 | Δ |
|:---|:---|:---|:---|:---|
| NASDAQ / NYSE / S&P500 | IC / RIC / Precision@N / SR | LSTM / ALSTM / RGCN / GAT / RSR-I / STHAN-SR / ESTIMATE / Linear | 未披露 | 平均相對性能提升 7.6%，10.8% 和 10.9%（p < 0.01） |

**解讀:** Δ 欄的 7.6%，10.8% 和 10.9% 反映模型在排名與風險調整回報上的真 capability，源於掩碼機制對時間偏差的魯棒性與超圖解耦對截面噪音的過濾。但此 Δ 極可能未計入交易成本、滑點與流動性限制；T=16 的短窗口與 2013-2017 美股低波環境暗示結果存在樣本內過擬合與前瞻偏差風險。基線模型逐一列出，未合成假 SOTA 區間。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations:** 在 NYSE（擁有最多的股票 1737）上觀察到性能輕微下降，推測候選池過大時歸納偏好不足逐漸發揮作用。超參數 m 與 k 需網格搜索，未自動化。
**6.2 推斷的隱含假設:** 
- **Regime 依賴:** 依賴穩定市場結構（2013-2017 長牛/低波），固定權重無法適應宏觀跳躍或流動性枯竭。
- **容量/成本:** 忽略實盤交易成本與漲跌停限制，純 MLP 推理雖快但缺乏訂單簿微結構感知。
- **數據泄漏:** 滑動窗口生成樣本若未嚴格隔離訓練/測試集時間邊界，易產生隱性前瞻偏差。
- **Survivorship:** 過濾小盤股與異常模式可能引入倖存者偏差，低估尾部風險。

🎤 **Interview Tip:** 
- **正確答:** 「StockMixer 的核心不是 MLP 本身，而是用矩陣掩碼強制時間因果序，並用超圖式降維壓縮截面交互，本質是將 GNN 的消息傳遞解耦為可並行的 MLP 操作，以換取訓練穩定性與低計算開銷。」
- **錯答:** 「它只是把 Transformer 的注意力換成了全連接層，所以跑得快。」（忽略了掩碼機制與超圖解耦的結構設計意圖）
**7.1 可證偽預測帶日期:** 若將該架構直接遷移至 A 股日频數據（高頻漲跌停限制與政策驅動 regime），預期 IC 與 SR 將在 2025 Q1 前顯著劣於 LSTM 基線，因固定掩碼無法捕捉政策跳躍帶來的非連續截面重定價。

## §8 · For the Reader
- **因子研究員:** 可將時間混合塊的輸出視為「動態截面表征」，直接接入傳統因子組合模型，替代手工因子；但需對多尺度 patch 做正交化處理以防多重共線性。
- **高頻執行:** 純 MLP 推理延遲極低，適合盤中動態調倉，但必須補入訂單簿微結構特徵與盤口失衡指標，否則無法應對流動性衝擊。
- **組合配置:** 利用其 Ranking Loss 輸出的排序信號做 Top-N 輪動，但必須疊加波動率過濾與尾部風險對沖，否則在 regime shift 事件中回撤不可控。
- **研究學生:** 復現門檻低，極佳的多尺度時間編碼與超圖交互的教學案例；建議嘗試將硬掩碼改為可學習的軟掩碼，並引入宏觀狀態變量做條件路由。
