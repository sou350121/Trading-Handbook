# CSHT 解構（CSHT）

> **發布**：2025-10-09 · （無 venue）
> **QuantML 導讀**：[剑桥 | 从新闻到回报：因果球体超图Transformer](https://mp.weixin.qq.com/s?__biz=Mzg2MzAwNzM0NQ==&mid=2247491913&idx=1&sn=b53dfc83a60ad9c64f7161490835e232&chksm=ce7d8657f90a0f41e6e8cb7ce591d94de01ea63488ff941a54d9622e90a07e3acbe15a9cf38b#rd)
> **核心定位**：落點於「因果結構×端到端表徵」的日頻波段框架。解決傳統多模態模型將新聞/情緒視為靜態黑箱輸入的 prior gap，透過格蘭傑因果超圖與測地線掩碼，強制注意力沿經濟學邏輯路徑流動。

**Status:** v0.5 — 基於 QuantML 導讀 + 原論文（如有）。benchmark 細節待升 v1。
**TL;DR:** ① 將新聞語義與情緒指標建模為有向超圖節點，以格蘭傑因果檢驗取代靜態相關性邊。② 核心 trick 是節點嵌入超球面後，用測地線掩碼約束 Transformer 注意力，僅允許沿真實因果父節點計算權重。③ 對「因果結構」軸而言，它將黑箱注意力轉為可追蹤的宏觀傳導鏈，直接對沖因子挖掘中的虛假相關風險。④ 導讀給出因果對齊度達 66.4%（遠超 FinGAT 的 21.3%），且 MAE 降低 13.4%。

**X-Ray.** 在「準確率 vs 可解釋性」的 Pareto 邊界上，CSHT 選擇以結構化掩碼換取推理空間的降維。它解決了舊工程坑：多模態輸入常因無約束注意力產生跨資產/跨時間的虛假關聯（causal leakage）。然而，其 envelope 受限於日頻波段與格蘭傑檢驗的線性預設，無法捕捉高頻微結構跳躍或非線性突變。對量化讀者而言，此架構不直接產出交易信號，而是提供一套「因果路徑驗證器」；實盤中可作為因子過濾層或風險歸因模組，但需警惕滑動視窗更新帶來的圖構建延遲與 regime 切換初期的信號滯後。

## §1 · 架構 / Core Mechanism
**1.1 三大改動 vs 前作**
| 維度 | 傳統 GNN / Transformer | CSHT | 工程意義 |
|---|---|---|---|
| 圖結構 | 靜態相關性邊 / 全連接 | 格蘭傑因果有向超邊 | 過濾虛假共變量，降低圖稀疏化成本 |
| 嵌入空間 | 歐幾里得空間 | 超球面黎曼流形 | 角度敏感度提升，幾何約束防止特徵漂移 |
| 注意力機制 | 無約束點積 / 自回歸掩碼 | 測地線掩碼 + 因果父節點過濾 | 強制模型遵循經濟傳導路徑，提升歸因透明度 |

**1.2 ⚡ Eureka 一句話 trick + 直覺**
「用格蘭傑檢驗畫出因果路徑，再把節點壓進球面，讓注意力只能沿測地線走。」直覺上，這相當於在 Transformer 內部硬編碼了一套宏觀經濟傳導邏輯，避免模型在噪音中過擬合。

**1.3 信息流 ASCII 圖**
```
新聞嵌入 (FinBERT) → 情緒聚合 (平滑) → [格蘭傑檢驗] → 有向超圖 G → 超球面嵌入 → 測地線掩碼 Attention → 預測頭 (回報/狀態)
```

## §2 · 數學層
📌 **Napkin Formula**：
`α_{i,j} = softmax(cos(u_i, u_j) / √d)  if j ∈ Pa_Granger(i)`
`α_{i,j} = 0                              otherwise`
**複雜度**：標準 Transformer O(N²) 注意力因因果掩碼大幅稀疏化，實際 FLOPs 取決於超邊平均度數。訓練採用黎曼優化：梯度在歐氏空間計算，更新後立即投影回單位超球面 (`u ← u / ||u||`)。
**直覺**：餘弦相似度在球面上等價於測地線距離，天然契合角度敏感的金融信號；掩碼機制將注意力分數從「數據驅動」轉為「結構驅動」，損失函數通常為回歸 MSE 或分類 Cross-Entropy，無額外正則項（結構本身即為歸納偏置）。

## §3 · 數據層
**資料規模/頻率/市場/時段**：2018-2023 年，450 只標普 500 成分股，日頻。
**怎麼來**：新聞標題經 FinBERT 編碼為嵌入向量；情緒得分由歷史新聞聚合平滑；市場特徵含對數回報、30 日已實現波動率、交易量。
**樣本外與容量假設**：訓練集 2018-2020，驗證集 2021，測試集 2022-年中 2023。容量假設受日頻限制，單日推理延遲 2.1ms，訓練時間隨資產數近線性增長，適合中低頻組合配置，不支撐高頻執行。

## §5 · 評測 / Benchmark
| 數據集/市場 | Metric | 前SOTA (FinGAT) | 本方法 (CSHT) | Δ |
|---|---|---|---|---|
| 450 S&P500 / 日頻 | MAE (回歸) | 未披露 | 未披露 | -13.4% |
| 450 S&P500 / 日頻 | Accuracy (3日分類) | 未披露 | 未披露 | +3.3pp |
| 450 S&P500 / 日頻 | NDCG@10 | 未披露 | 未披露 | 最高分 |
| 450 S&P500 / 日頻 | Causal Alignment | 21.3% | 66.4% | +45.1pp |
| 2020.03-06 / 疫情衝擊 | Accuracy | 未披露 | 74.6% | 顯著優於基線 |

**解讀**：Δ 中的 +45.1pp 因果對齊度與 -13.4% MAE 屬真 capability，證明結構掩碼有效過濾虛假依賴；但 Accuracy +3.3pp 與 NDCG@10 優勢可能部分來自樣本外時段（2022-2023）的特定波動率環境，未計入交易成本與滑點前，實盤 Sharpe 可能收斂。疫情期間 74.6% 準確率反映模型對極端外生衝擊的韌性，但格蘭傑檢驗的滯後性可能導致 regime 切換初期的信號延遲。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations**：導讀未明確列出 limitations。
**6.2 推斷的隱含假設**：
- **Regime 依賴**：滑動視窗更新圖結構，假設因果關係變化緩慢；若遇政策急轉或流動性枯竭，圖更新滯後將導致掩碼失效。
- **數據泄漏風險**：情緒得分由「過去新聞」聚合，若聚合窗口與預測目標重疊，可能引入輕度前瞻偏差。
- **線性預設**：格蘭傑因果本質為線性預測檢驗，無法捕捉非線性閾值效應或高階交互作用。
- **容量/成本**：日頻推論雖快（2.1ms），但圖構建與黎曼投影增加訓練開銷；實盤需評估數據授權成本與每日重算超圖的算力邊界。

## §8 · For the Reader
- **因子研究員**：將 CSHT 的因果對齊度作為因子穩定性指標；若某因子在超圖中無法形成顯著格蘭傑路徑，應降權或剔除。
- **組合配置**：利用 NDCG@10 排序結果構建多空組合，但需疊加波動率目標與行業中性約束，避免過度集中於科技/金融板塊。
- **LLM-Agent / 研究學生**：將 FinBERT 嵌入替換為最新 LLM 金融微調版本，測試語義密度提升是否會破壞原有球面幾何約束；可作為論文複現與架構擴展的起點。
