> **核心定位**：落點於「文本另类×日频波段×监督回归」，以 FinBERT 提取宏觀新聞極性，透過 XGBoost 捕捉非線性交互，並用 SHAP 解構決策路徑。解決了傳統詞典情緒模型在宏觀資產（FX/債券）上的語境誤判與黑箱合規痛點。

**Status:** v0.5 — 基於 QuantML 導讀 + 原論文（如有）。benchmark 細節待升 v1。
**TL;DR:** ① 將 GDELT 宏觀新聞流經 FinBERT 轉為極性分數，構建日頻情緒特徵集。② 核心 trick 在於結合情緒分散度（sentiment_std）與文章影響（article_impact）等時序衍生特徵，以 XGBoost 擬合次日方向，並用 SHAP 實現特徵歸因透明化。③ 對「因子挖掘×可解釋性」軸★，提供了一套開源、可復現且合規友好的宏觀 Alpha 生成範式。④ 樣本外夏普比率達 EUR/USD 5.87 / USD/JPY 4.65 / ZN 4.65（已扣成本）。

**X-Ray.** 放回五軸 Pareto：在「數據模態」上從粗糙詞典躍遷至領域預訓練 Transformer，但代價是推理延遲與 API 依賴；在「時間尺度」上鎖定日頻波段，避開了高頻數據的雜訊與成本陷阱，但依賴新聞發佈與市場定價的日級傳導滯後。解了舊工程坑：傳統情緒策略常因前瞻偏差與過擬合在 OOS 崩盤，本文以 5 折擴展窗口 + 特徵熱身期 + 嚴格收盤後數據切割，硬鎖了數據泄漏路徑。預測打不開的 envelope：策略高度依賴宏觀新聞流的「分歧度」與「影響加權」，在低波動/無新聞 regime 下信號會退化為隨機游走；且 SHAP 僅提供局部線性近似，無法捕捉 XGBoost 深層樹結構的動態交互。對量化讀者意義：提供了一個「可審計」的因子挖掘 pipeline，證明開源 NLP + 樹模型在宏觀資產上仍能產出高 Sharpe，但需警惕 5.87 的 Sharpe 可能部分源於 2017-2025 特定宏觀週期（利率/地緣衝突頻發）的 regime 紅利，而非純粹的 alpha 穩定性。

## §1 · 架構 / Core Mechanism
| 改動維度 | 前作/傳統基線 | 本法改動 | 工程/統計意圖 |
|:---|:---|:---|:---|
| 情緒提取 | 粗糙詞典/通用模型 | FinBERT（金融預訓練） | 捕捉宏觀術語語境，降低誤分類率 |
| 特徵構造 | 單一日平均極性 | 分散度、加權影響、滯後/滑動統計 | 將新聞流轉為動量與分歧信號，擬合非線性 |
| 決策透明 | 黑箱樹模型/深度網絡 | SHAP 歸因 + 邏輯迴歸對照 | 滿足合規審計，驗證特徵經濟直覺一致性 |

⚡ **Eureka:** 情緒的「分歧度」（sentiment_std）與「影響加權」（article_impact）比單純的「平均極性」更具預測力，市場對「衝突敘事」的定價效率低於「共識敘事」。

```
GDELT API (事件碼 100-199)
   ↓ 每日 Top 100 標題 + Goldstein 權重
FinBERT (ProsusAI/finbert) → 連續極性分數
   ↓ 特徵工程 (MA/滾動Std/滯後/波動率)
XGBoost (二元分類) → 次日方向概率
   ↓ 閾值 0.5 切分 → 信號 (+1/-1)
SHAP 歸因 → 特徵貢獻可視化/合規報告
```

## §2 · 數學層
📌 **Napkin Formula:**
`polarity_t = (P_pos - P_neg) / (P_pos + P_neutral + P_neg)` （公式截斷，直覺為淨看漲比例）
`L = -[y log(p) + (1-y) log(1-p)]` （二元邏輯損失）
複雜度：`O(N * trees * depth)`，訓練依賴 5 折擴展窗口與早停。

**直覺:** 將離散新聞流壓縮為連續極性序列，透過時序滑窗捕捉情緒動量與均值回歸傾向。XGBoost 自動擬合極性與新聞量/波動率的交互項，SHAP 則將樹節點分裂路徑拆解為加性貢獻，使非線性決策可追溯。

## §3 · 數據層
- **規模/頻率/市場/時段:** 日頻。`2015年1月1日` 至 `2025年4月30日`。目標：EUR/USD, USD/JPY, 10Y 美債期貨（ZN）。
- **來源:** GDELT v2 API（新聞標題+Goldstein量表）, Yahoo Finance（收盤價/連續合約滾動）。
- **樣本外與容量假設:** OOS 嚴格切割為 `2017年初` 至 `2025年4月30日`（5 折擴展窗口）。容量假設未披露，但 FX/ZN 日頻流動性高，理論可承載機構級資金；策略未計滑點與衝擊成本，實際容量取決於執行算法。

**解讀:** Δ 欄為 `未披露` 因導讀未提供基線具體數值。本方法數值顯著高於傳統線性基線（邏輯迴歸在債券上「掙扎」）。`5.87` 與 `4.65` 的差異反映外匯與債券的內在波動性與機會集不同。該 Δ 主要來自 XGBoost 對非線性交互（情緒分歧×新聞量）的擬合能力，但需警惕高 Sharpe 可能部分源於 `2017年初` 至 `2025年4月30日` 期間宏觀波動 regime 的 beta 溢價，而非純粹的 alpha 穩定性。成本已計（外匯 `0.02%` / ZN `0.05%`），未計滑點。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations:** LLM 在股票情緒上可能優於 FinBERT，但本研究聚焦宏觀與開源可復現性；外匯與債券 CAGR 差異源於資產內在波動性；SHAP 僅提供局部解釋，無法完全揭示深層樹結構交互。
**6.2 推斷的隱含假設:** 
- **Regime 依賴:** 策略依賴新聞流的高頻與高分歧度，在「低波動/無重大宏觀事件」 regime 下，`sentiment_std` 權重將失效，信號退化為隨機游走。
- **數據泄漏防線:** 嚴格依賴 `2015年1月1日` 至 `2025年4月30日` 的收盤後數據切割與特徵熱身期，假設 GDELT 事件碼 `100-199` 過濾邏輯能持續捕捉宏觀相關性。
- **成本/執行:** 假設 `0.02%` / `0.05%` 的單邊成本可穩定執行，未考慮流動性枯竭或極端行情下的滑點膨脹。Block bootstrap（`1,000` 次重採樣，塊大小 `20`）假設日回報自相關結構在樣本外穩定。

🎤 **Interview Tip:** 
- **正確答:** 「本框架的核心價值不在於單純的預測精度，而在於透過 SHAP 將 XGBoost 的非線性決策映射回經濟直覺（如情緒分歧度定價），滿足合規審計需求。高 Sharpe 需結合 5 折擴展窗口與嚴格 OOS 切割解讀，避免將 regime beta 誤認為穩定 alpha。」
- **錯答:** 「FinBERT 比所有 LLM 都好，SHAP 能完全解釋黑箱模型，策略在任何市場環境下都能保持 5.87 的夏普。」

**7.1 可證偽預測:** 若 `2025Q3` 宏觀新聞量驟降或地緣衝突平息，`sentiment_std` 特徵權重將失效，策略 OOS Sharpe 預期回落至 `<3.0`（導讀未給具體閾值，僅作結構性推斷）。

## §8 · For the Reader
- **因子研究員:** 將 `sentiment_std` 與 `article_impact` 視為獨立因子池，測試其與傳統宏觀因子（期限利差、匯率動量）的正交性，避免特徵共線性稀釋 Alpha。
- **組合配置:** 該策略提供日頻方向信號，適合與低頻資產配置模型（如風險平價）疊加；需監控 `0.02%`/`0.05%` 成本在組合層級的累積摩擦。
- **LLM-agent/自動化研究:** 利用本 pipeline 的開源特性，將 FinBERT 替換為最新開源 LLM（如 Qwen/Llama 金融微調版），透過 SHAP 對比歸因穩定性，驗證「模型規模 vs 可解釋性」的 Pareto 邊界。
