> **核心定位**：將因子篩選從「靜態數值回歸」轉為「語義條件門控」，用 GRPO 對齊 8B 推理模型，以價格與新聞雙模態上下文動態激活/停用因子，直擊非平穩市場中的模型設定錯誤（Model Misspecification）與 Alpha 衰減。

**Status:** v0.5 — 基於 QuantML 導讀 + 原論文（如有）。benchmark 細節待升 v1。
**TL;DR:** ① 用 GRPO 將 8B LLM 對齊至客觀市場回報，取代傳統靜態權重或純數值 RL。② 核心 trick 是「雙層語義上下文」+「條件門控」：LLM 讀懂因子失效條件與當前新聞/價格狀態，按需開關因子。③ 對因子挖掘軸★：把非平穩性適配從「參數重估」外包給「語義推理」，降低過擬合風險。④ 域內 CSI 300 夏普比率 1.62 / 最大回撤 6.76%，域外 CSI 1000 零樣本夏普比率 4.03。

**X-Ray.** 傳統多因子框架的致命傷不在估計誤差，而在 Regime Shift 下的 Model Misspecification。Alpha-R1 的 Pareto 落點很明確：放棄純數值特徵工程，將「何時該用哪個因子」的決策權交給具備經濟邏輯推理能力的 LLM。它解了兩個舊坑：一是靜態權重/樹模型在風格切換時對噪聲過度反應（如導讀指出 XGBoost 累積回報 -10.03%）；二是通用推理 LLM 缺乏金融基礎導致風控崩潰（回撤約15-16%）。然而，該架構打不開的 Envelope 同樣清晰：GRPO 依賴固定線性評分器提供穩定獎勵，這意味著它本質上仍是「線性因子模型 + 語義開關」，無法捕捉因子間的高階非線性交互；且新聞語義的提取質量高度依賴外部 LLM 裁判與預設的結構化模板，在極端流動性枯竭或政策黑天鵝期間，語義滯後性將直接轉化為門控延遲。對量化讀者而言，其價值不在於直接實盤，而在於提供了一套「可解釋的動態因子權重分配」範式，可與傳統風險模型或高頻執行層解耦組合。

## §1 · 架構 / Core Mechanism
**1.1 三大改動 vs 前作**
| 維度 | 傳統靜態/數值 RL | 通用推理 LLM | Alpha-R1 |
|---|---|---|---|
| 決策核心 | 歷史相關性/數值梯度 | 通用邏輯推理 | 語義條件門控 (Context-Conditioned Gating) |
| 優化目標 | 預測誤差/純數值回報 | 人類偏好/代碼正確率 | 客觀市場回報 + 推理連貫性 (GRPO) |
| 狀態表示 | 數值特徵向量 | 純文本 Prompt | 雙層語義上下文 (長期市場記憶 + 即時價格/新聞) |

**1.2 ⚡ Eureka 一句話 trick + 直覺**
Trick: 用 GRPO 的無 Critic 架構，將「因子激活/停用」建模為序列決策，獎勵信號由固定線性模型的回測超額回報與外部 LLM 裁判的邏輯一致性評分拼接而成。
直覺: 不教模型「怎麼算因子」，只教模型「什麼市場環境下因子會失效」，讓 LLM 當門衛而非算盤。

**1.3 信息流 ASCII 圖**
```
[原始價格/新聞] -> [原子文本化] -> [迭代記憶構建(週)] -> [長期市場描述 G]
[因子池回測] -> [定量表現向量 V] -> [LLM語義剖析 S]
[即時狀態合成] -> [資產池狀態描述 D_t]
[G + S + D_t] -> [語義決策上下文 C_t] -> [Alpha-R1 推理] -> [選定因子列表 F_t]
[F_t] -> [固定線性評分器] -> [股票排名] -> [TopN等權組合] -> [GRPO獎勵反饋]
```

## §2 · 數學層
📌 **Napkin Formula**:
獎勵函數: $R = R_{perf} + R_{quality} + R_{struct}$
線性評分: $\hat{r}_i = \sum_{j \in F_t} \beta_j \cdot x_{i,j}$ (係數 $\beta$ 固定)
GRPO 優勢估計: $\hat{A}_{i,t} = \frac{R(o_i) - \text{mean}(R(o_{group}))}{\text{std}(R(o_{group}))}$
複雜度: 推理階段 $O(N_{factors} \cdot L_{context})$，訓練階段依賴 GRPO 群組採樣，無 Critic 網絡降低顯存開銷。
直覺: 固定 $\beta$ 確保映射穩定，LLM 只輸出稀疏子集 $F_t$。GRPO 用群組內相對排名替代絕對價值估計，規避了金融數據非平穩性導致的 Critic 崩潰。
Loss/訓練細節: 截斷代理目標 + KL 散度正則化，防止偏離 Qwen3-8B 預訓練分佈。

**解讀**: Δ 的真實能力來源於「語義門控」對非平穩性的適配，而非參數過擬合。消融實驗顯示，去除 RL 對齊後夏普比率降至 -0.77，證明純基座模型無法將語義知識轉化為風控行動；去除新聞語義導致夏普比率降至 1.03，證實定性信號在風格切換檢測上的領先性。需警惕的偏差：固定線性評分器在 RL 過程中提供穩定獎勵，這可能掩蓋了因子間真實的非線性交互；且 VWAP 執行與 0.1% 雙邊成本雖已計入，但未披露滑點模型與衝擊成本，實盤 Δ 可能收斂。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations**: 依賴外部 LLM 裁判進行質量調整獎勵，可能引入裁判模型的主觀偏差與 API 延遲；固定線性評分器雖穩定，但無法捕捉因子高階交互；新聞數據的語義提取存在滯後性。
**6.2 推斷的隱含假設**:
- Regime 依賴: 假設市場風格切換會先反映在新聞敘事與宏觀公告中，若切換由突發流動性衝擊驅動，語義門控將失效。
- 容量/成本: Slot Rotation 機制將平均日換手率降低至 TBD，假設 VWAP 前 30 分鐘流動性充足；未披露極端行情下的流動性假設。
- 數據泄漏: 預訓練階段使用 2020.01-2023.12 估計固定係數，若該區間包含未來信息或因子計算存在前視偏差，獎勵信號將失真。
- Survivorship: 未明確說明是否處理退市/停牌股票的生存者偏差，CSI 1000 零樣本測試若未過濾流動性極差標的，夏普比率 4.03 可能虛高。

## §8 · For the Reader
- **因子研究員**: 別再死磕因子 IC 衰減曲線。將你的因子邏輯轉為結構化語義描述（機制/失效條件），接入此框架做動態權重分配，可顯著降低 Regime Shift 下的回撤。
- **高頻執行/組合配置**: 該框架輸出的是日頻因子子集，需與你的執行算法（如 VWAP/POV）解耦。注意 Slot Rotation 機制可平滑換手，但實盤需自測衝擊成本對 0.1% 假設的偏離。
- **LLM-Agent/RL 策略**: GRPO 在此的應用示範了如何將「客觀市場回報」轉化為 RL 獎勵。若你正在做金融 Agent，可借鑒其「無 Critic + 群組相對優勢 + 結構懲罰」的穩定訓練范式，避免 PPO 的顯存瓶頸。
