# FRA特征筛选算法 解構（FRA特征筛选算法）

> **發布**：2025-06-29 · （無 venue）
> **QuantML 導讀**：[基于多数据源的加密货币预测](https://mp.weixin.qq.com/s?__biz=Mzg2MzAwNzM0NQ==&mid=2247490876&idx=1&sn=725f4edfb65a648fcdc5397d6b3a43ee&chksm=ce7e7a22f909f33499603147b700f73057d55dc071b0a3e863c64a84ec024c24cf6abccfba7d#rd)
> **核心定位**：落點於「多模態×跨周期」的因子挖掘與可解釋性交叉帶。解決了加密預測長期依賴單一技術指標/情緒指標、缺乏系統性跨源驗證與動態特徵降維的工程痛點。

**Status:** v0.5 — 基於 QuantML 導讀 + 原論文（如有）。benchmark 細節待升 v1。
**TL;DR:** ① 提出FRA迭代特徵篩選算法，系統量化五大類數據源在1-180天預測窗口的貢獻度。② 核心trick是動態閾值遞增的聯合剪枝機制，融合SHAP/PFI/MDI/皮爾遜四維驗證。③ 對「因子挖掘」軸★，它提供了一套可審計的跨周期特徵圖譜，直接對沖多模態數據的共線性與過擬合風險。④ 關鍵實證數字：多樣化數據源使RF模型MSE平均下降455.67%（2017集）與426.67%（2019集）。

**X-Ray.** 將FRA放回五軸Pareto，其本質是「監督回歸」範式下的特徵工程自動化，而非端到端黑盒。它解了兩個舊坑：一是單一重要性指標在共線性強的多模態數據中易產生虛假排名；二是跨周期預測中特徵權重的靜態假設。FRA透過動態閾值遞增與多模型交叉驗證，強制保留跨週期穩定的「基本面因子」，過濾短期噪聲。預測其打不開的envelope在於：算法仍依賴樹模型與線性相關性，對高頻非線性交互與深度時序依賴捕捉有限。對量化讀者意義在於，它提供了一套可解釋的特徵降維流水線，可直接嵌入因子庫構建，但需警惕其「多樣性提升」部分來自基線單一類別特徵的極端低效，實盤需計入數據獲取與清洗成本。

## §1 · 架構 / Core Mechanism
**1.1 三大改動 vs 前作**
| 維度 | 前作/常規做法 | FRA算法改動 | 工程意圖 |
|---|---|---|---|
| 特徵評估標準 | 單一指標（如僅SHAP或僅PFI） | 融合MDI/PFI/皮爾遜/SHAP四維交叉驗證 | 規避單一評估方法的偏差與共線性干擾 |
| 降維策略 | 靜態閾值截斷或逐步回歸 | 動態閾值遞增的迭代剪枝（底層50%聯合移除） | 適應多模態數據分佈漂移，強制收斂至穩定子集 |
| 週期假設 | 單一預測窗口獨立建模 | 跨周期（1d-180d）貢獻率動態追蹤 | 揭示短/長週期驅動因子切換，支援多頻策略組合 |

**1.2 ⚡ Eureka 一句話 trick + 直覺**
Trick: 每輪迭代同時計算四種重要性，鎖定始終墊底50%的特徵，並配合皮爾遜閾值從0.5起每輪遞增0.025的聯合剪枝。
直覺: 不依賴任何單一模型的「偏見」，用動態收緊的線性約束過濾掉在多種評估體系下均不穩定的噪聲特徵，直到維度壓至100。

**1.3 信息流 ASCII 圖**
```
[多模態原始數據] → [Crypto100指數標註] → [樹模型(MDI/PFI) + 線性(皮爾遜)]
          ↓
[FRA迭代剪枝] → (底層50%標記 + 動態閾值遞增0.025) → [剩餘<100特徵]
          ↓
[SHAP獨立驗證] → (Top100重疊率78) → [FRA Top75 ∪ SHAP Top75] → [最終特徵向量]
```

## §2 · 數學層
📌 **Napkin Formula:**
`FRA_t = { f_i | rank_MDI(f_i) > 50% ∧ rank_PFI(f_i) > 50% ∧ |ρ(f_i, y)| > τ_t }`
`τ_{t+1} = τ_t + 0.025`, 初始 `τ_0 = 0.5`, 終止條件 `|FRA_t| < 100`
**直覺:** 聯合排序閾值過濾，動態收緊線性約束防止特徵集膨脹。
**Loss/訓練:** 監督回歸（預測Crypto100價格），使用RF/XGB評估MSE下降，未披露具體正則化或學習率細節。

**解讀:** Δ 反映的是「多樣化特徵集」對比「單一類別特徵集」的內部對照提升，非對外SOTA超越。455.67%等巨幅提升主要源於基線單一類別（如純技術指標）在長週期預測中的極端低效，屬數據維度補全帶來的邊際收益，非算法本身創造的絕對Alpha。實盤需警惕：MSE下降未轉換為交易收益（Sharpe/IR），且未計入多源數據獲取、清洗與延遲成本；長週期（90天/180天）預測的MSE改善可能受宏觀趨勢順風驅動，存在前瞻偏差風險。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations:** 情緒與宏觀類別代表性不足可能低估其重要性；僅用BTC與USDC鏈上數據缺乏DeFi維度；未驗證對深度學習模型的增益或噪聲引入。
**6.2 推斷的隱含假設:** Regime依賴強（2017集與2019集結果差異顯示牛熊週期切換影響特徵穩定性）；容量假設未披露，但日頻Crypto100預測適合中低頻配置；數據泄漏風險存在（特徵計算可能隱含未來信息，如某些鏈上指標的確認延遲）；Survivorship bias未明確處理（Crypto100動態排名可能遺棄已歸零代幣）。

🎤 **Interview Tip:** 
正確答: FRA的核心不在於單一指標的精度，而在於透過動態閾值與多模型交叉驗證，強制篩選出跨週期穩定的特徵子集，降低多模態數據的共線性與過擬合風險。實盤應用需將MSE提升轉換為風險調整收益，並計入數據成本。
錯答: FRA直接超越了所有現有預測模型，因為它用了SHAP和XGBoost，所以預測一定更準。

**7.1 可證偽預測:** 若將FRA應用於2024-2025年加密市場，其保留的「傳統市場指數」與「USDC鏈上指標」在長期窗口的貢獻率將因監管政策與穩定幣市場結構變化而顯著下降（預測日期：2025-12-31前驗證）。
