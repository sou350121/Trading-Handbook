**TL;DR:** ① 利用 LLM 文本嵌入構建動態同業網絡，量化盈余驚喜溢出效應。② 核心 trick 是將公告公司 SUE 按余弦相似度加權投影至非公告公司，生成 Proj SUE。③ 對「因子挖掘」軸★，將橫截面異象從風險補償敘事轉為信息擴散誤定價，提供可解釋的動態因子構建路徑。④ 導讀給出多空策略年化收益率 11%，夏普比率 1.75。

**X-Ray.** 本方法在五軸 Pareto 前沿上，用語義距離替換了靜態行業代碼，解決了量化工程中「同業定義滯後」與「信息路徑模糊」兩大坑。傳統因子依賴財務特徵或供應鏈圖譜，更新頻率低且難以量化無形資產與戰略轉向；Proj SUE 將盈餘電話會議轉為 3072 維向量，以余弦相似度作為鄰接矩權重，實現了日頻維度的動態重構。預測其打不開的 envelope 在於：① 依賴管理層陳述的語義穩定性，若宏觀 regime 切換導致話術框架崩解（如政策急轉或會計準則變更），嵌入空間的局部結構一致性將失效；② 20 個交易日的價格漂移暗示 alpha 衰減緩慢，策略容量受限，大資金難以在日頻重倉下維持 1.75 的 Sharpe；③ 對量化讀者的意義不在於直接抄因子，而在於示範了如何將 LLM 嵌入轉化為可交易的圖加權截面信號，並提供了一套驗證「信息擴散 vs 板塊 Beta」的固定效應控制範式。

## §1 · 架構 / Core Mechanism
| 維度 | 前作/傳統做法 | 本方法 (Proj SUE) | 工程意義 |
|:---|:---|:---|:---|
| 同業定義 | SIC 代碼 / 供應鏈靜態關係 | LLM 文本嵌入 (3072 維) 余弦相似度 | 突破靜態分類，捕捉業務重心轉移與技術變革 |
| 信號構建 | 單公司 SUE 橫截面排序 | 公告公司 SUE 加權投影至非公告公司 | 將孤立異象轉化為網絡溢出因子，提升信號覆蓋面 |
| 異象解釋 | 風險補償 (CAPM/FF 因子) | 信息擴散誤定價 + 動態定價因子 | 解釋 2009 年後多數頑固異象，提供因果鏈條驗證 |

⚡ **Eureka 一句話 trick**：用盈餘電話會議的語義距離作為鄰接權重，將公告公司的 SUE 投影到非公告公司，構建日頻更新的溢出因子。
**直覺**：投資者注意力有限，同行財報的超預期信息會沿業務關聯網絡緩慢擴散，價格調整滯後形成可捕獲的漂移。

```
盈餘電話會議文本
       ↓ (OpenAI text-embedding-3-large)
3072維語義向量 (季度更新)
       ↓ (余弦相似度計算)
動態同業鄰接矩陣 W_{i,j}
       ↓ (加權投影)
ProjSUE_{i,t} = Σ_j (SUE_{j,t} × W_{i,j})
       ↓ (截面排序)
做多 Top Quintile / 做空 Bottom Quintile (剔除公告公司)
       ↓ (日頻重倉)
Alpha 提取 + 20個交易日漂移監控
```

## §2 · 數學層
📌 **Napkin Formula**：
`ProjSUE_{i,t} = Σ_{j∈Announcing} SUE_{j,t} × cos(Emb_{i,t-1}, Emb_{j,t-1})`
**複雜度**：日頻投影 O(N·M)，M 為當日公告公司數；嵌入計算 O(T·D) 按季度批處理。
**直覺**：SUE 定義沿用 Livnat 和 Mendenhall（2006）方法（實際 EPS 與分析師預測中位數差值經股價標準化）。投影本質是圖濾波器，將局部超預期信號平滑至語義鄰域。
**Loss/訓練細節**：無自定義損失函數。採用凍結的 OpenAI text-embedding-3-large 模型，依賴預訓練語義空間的零樣本泛化能力，僅在下游進行相似度計算與截面排序。

**解讀論斷**：
- 每日 4-5 bps 的 Alpha 在控制日期/股票/行業×日期固定效應後依然顯著，且 75% 來自行業內部差異化信息，說明信號具備真實的橫截面定價能力，而非板塊 Beta 漂移。
- 1.75 的 Sharpe 與 11% 年化為毛收益（導讀未披露交易成本與滑點）。日頻重倉 + 20 個交易日漂移意味著策略對執行成本極度敏感，實盤 Sharpe 大概率收斂。
- 異象定價結果（28→23）證明 Proj SUE 捕獲了傳統模型遺漏的信息擴散路徑，但剩餘的投資增長與研發市值比因子仍有殘餘 Alpha，暗示該方法對資本支出週期長的行業覆蓋不足。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations**：LLM 訓練階段可能見過樣本期內數據（Data Leakage）；研究在 2024年樣本進行純樣本外測試，預測能力依然穩健，排除單純模型記憶。
**6.2 推斷的隱含假設**：
- **Regime 依賴**：信號強度高度集中於財報季，非財報期信號稀疏，需配合波動率過濾或動態倉位管理。
- **容量約束**：20 個交易日的價格漂移表明市場消化緩慢，大資金難以在日頻維度滿倉運作，策略容量受限。
- **成本未計**：每日調倉 + LLM API 嵌入成本未納入績效；若單筆交易成本超 3 bps，多空淨值將受侵蝕。
- **數據泄漏風險**：語義空間可能隱式吸收宏觀新聞或市場情緒，需嚴格控制新聞共現網絡與嵌入空間的因果隔離。
- **幸存者偏差**：導讀未明確標註退市公司處理方式，若僅覆蓋存活公司，Alpha 可能被高估。

## §8 · For the Reader
- **因子研究員**：將 Proj SUE 視為動態圖加權截面信號，替代傳統 SIC 排序。重點驗證嵌入空間與財務特徵空間的互 K 近鄰一致性，避免語義漂移。
- **高頻執行**：20 個交易日漂移意味著 alpha 衰減緩慢，日頻重倉需精確建模滑點與衝擊成本。建議採用分批建倉 + 波動率縮減倉位。
- **組合配置**：將 Proj SUE 作為動態風險因子疊加至現有模型。其對價值與規模因子的短期預測力可用於戰術倉位調整，但需監控財報季外的信號稀疏期。
- **LLM-agent**：關注 prompt 工程對嵌入空間局部結構的影響。可嘗試引入行業術語字典或宏觀事件標簽進行空間對齊校準，提升 mNN 指標穩定性。
