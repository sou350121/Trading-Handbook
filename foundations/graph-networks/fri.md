# FRI框架 解構（FRI框架）

> **發布**：2025-09-25 · （無 venue）
> **QuantML 導讀**：[面向解释性的金融图谱量化框架](https://mp.weixin.qq.com/s?__biz=Mzg2MzAwNzM0NQ==&mid=2247491788&idx=1&sn=f808ef78c367f774d0195fc7ec50d7a8&chksm=ce7d87d2f90a0ec448c30728663ec40b3904e83e4859248445e50572367c3a2ab5e3fd669de2#rd)
> **核心定位**：落點於「图关系」與「人机协同可解释」軸，解決了傳統 GNN 因子研究將「圖結構質量」與「下游預測指標」深度綁定所導致的評估混淆與過擬合先驗缺口。

**Status:** v0.5 — 基於 QuantML 導讀 + 原論文（如有）。benchmark 細節待升 v1。
**TL;DR:** ① 提出獨立於下游任務的 FRI 評估框架與 SPNews 數據集，實現動態金融圖譜的質量量化。② 核心 trick 為「先解釋後預測」，透過收益相關性穩定性、事件捕捉率等 4 項獨立指標驗證圖結構有效性。③ 此設計直接鬆綁了端到端表征對特定任務的依賴，使圖構造可被獨立審計。④ 導讀未給量化結果。

**X-Ray.** FRI 框架將圖神經網絡的「黑盒構造」拆解為可審計的「白盒指標」，在「人机协同可解释」軸上切開了傳統端到端表征的過擬合死結。它解決了量化工程中「圖結構優劣僅能靠下游回測倒推」的歸因模糊坑，但打不開「邊權重質量評估」與「跨市場泛化」的 envelope。對實盤研究員而言，此框架不直接產出 Alpha，而是提供圖構造階段的預篩選閾值，大幅降低特徵工程階段的算力浪費與前瞻偏差風險。

## §1 · 架構 / Core Mechanism
**1.1 三大改動 vs 前作**
| 維度 | 前作/傳統範式 | FRI 框架改動 |
|---|---|---|
| 關係模態 | 預設靜態關係（如行業分類）或單一量化信號 | 時序演變圖集合，基於新聞共現動態構建邊 |
| 評估邏輯 | 與下游預測任務（如股價預測）深度綁定 | 獨立於下游任務，以歷史金融現象解釋力為驗收標準 |
| 驗證路徑 | 模型損失倒推圖結構優劣 | 「先解釋後預測」，統計檢驗先行，下游任務後置 |

**1.2 ⚡ Eureka**
用歷史金融現象的統計顯著性反推圖結構有效性，而非用預測損失倒推。

**1.3 信息流 ASCII**
```
新聞語料 (SPNews) → 共現檢索/閾值τ → 動態邊存在性/強度 → 時序圖集合
       ↓
水平維度檢驗 (單期連接 vs 孤立節點金融特性差異)
垂直維度檢驗 (邊動態演變 vs 金融現象變化共振)
       ↓
FRI 四項獨立指標 (CSS / AECR / Δβ / Δγ) → 圖結構質量評分
       ↓
(可選) 下游 GNN 訓練與預測任務
```

## §2 · 數學層
📌 **Napkin Formula**
- `CSS = (1/T) Σ I(拒絕 H₀ at t)`，檢驗連接節點對相關性變化量是否顯著大於無連接節點對。
- `AECR = mean(I(事件期內相關性波動 > k·標準差))`，衡量邊動態與真實商業事件的吻合度。
- `Δβ = mean(β_{i+1} - β_i)`，關係因子對收益相關性的分組回歸係數增量。
- `Δγ = γ_high - γ_low` (基於 DCC-GARCH)，衡量波動率相關性對新信息敏感度與持續性的差異。
**複雜度**：`O(T · N²)`（窗口相關性計算與節點對分組回歸）。
**直覺**：水平維度檢驗單期圖內連接節點 vs 孤立節點的金融特性差異；垂直維度追蹤邊動態演變與金融現象變化的因果共振。
**Loss/訓練細節**：FRI 本身為統計檢驗與因子回歸框架，無傳統監督損失。下游任務使用 GAT+LSTM，損失函數與優化器細節導讀未披露。

## §5 · 評測 / Benchmark
| 數據集/市場 | Metric | 前SOTA/基線 | 本方法 | Δ |
|---|---|---|---|---|
| SPNews (美股) | AECR | StaticGraph 0.078 | DynamicGraphSPNews τ=0 最高 | 未披露 |
| SPNews (美股) | AECR | DynamicGraphCorr | DynamicGraphSPNews τ=0 最高 | 未披露 |
| SPNews (美股) | ACC / Macro F1 | StaticGraph | DynamicGraphSPNews τ=1 最佳 | 未披露 |
| SPNews (美股) | ACC / Macro F1 | DynamicGraphCorr | DynamicGraphSPNews τ=1 最佳 | 未披露 |

**解讀**：Δ 欄因導讀未給具體數值故寫未披露。靜態圖 AECR 為 0.078 屬明確披露值。FRI 排序與下游任務排序高度相關，證明其作為預篩選工具的有效性；但 τ=0 (FRI最佳) 與 τ=1 (下游最佳) 不完全一致，暗示圖結構解釋力與預測收益之間存在非線性映射。下游任務的 ACC/F1 提升可能部分來自 GAT+LSTM 的表徵能力而非純粹圖結構優化。未計入新聞抓取與共現計算的數據成本，實盤部署需評估延遲與算力開銷。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations**
- 三個指標不適用於靜態圖。
- 無法評估加權圖中邊的「權重」質量，僅能評估邊的存在與時間動態。

**6.2 推斷的隱含假設**
- **Regime 依賴**：指標有效性高度依賴新聞語義與市場情緒共振期，在無事件驅動的低波動 regime 下，CSS 與 AECR 可能退化為隨機噪聲。
- **容量/成本**：新聞共現檢索與 DCC-GARCH 分組回歸計算成本高，不適合全市場萬級標的實時運算。
- **數據泄漏**：新聞時間戳與股價反應存在天然延遲，若未嚴格對齊交易時段，垂直維度檢驗可能混入前瞻偏差。
- **Survivorship**：僅保留 431 只成分股，未處理退市/並購樣本，長期回測存在生存偏差。

🎤 **Interview Tip**
- **正確答**：FRI 是圖構造階段的質量審計工具，不替代下游預測模型，用於解耦圖結構優劣與模型容量，降低特徵工程階段的試錯成本。
- **錯答**：FRI 直接輸出交易信號或替代 GNN 訓練損失。

**7.1 可證偽預測帶日期**
若將 FRI 框架遷移至新聞語義噪聲較高或共現閾值 τ 敏感區間的其他市場，其 CSS 與 AECR 指標與下游任務的排序相關性將顯著下降（預測驗證期：TBD）。

## §8 · For the Reader
- **因子研究員**：用 FRI 的 Δβ 與 Δγ 替代傳統 IC/IR 進行圖特徵預篩選，砍掉無效邊構造實驗，聚焦高解釋力子圖。
- **高頻執行**：本框架為日頻波段設計，不適用 tick 級訂單流圖；若需高頻，需將新聞共現替換為訂單簿動態匹配與盤口共振檢驗。
- **組合配置**：將 FRI 評分作為圖結構正則化項，約束端到端表征的過擬合邊，提升組合在 regime shift 下的穩健性與風險預算分配效率。
- **LLM-agent/研究學生**：利用 SPNews 的標註結構訓練輕量級關係抽取模型，驗證大模型在金融實體鏈接中的零樣本泛化能力，並對接 FRI 指標進行自動化圖質量審計。
