# DeltaLag 解構（DeltaLag）

> **發布**：2025-11-09 · （無 venue）
> **QuantML 導讀**：[DeltaLag: 挖掘动态领涨-滞涨Alpha](https://mp.weixin.qq.com/s?__biz=Mzg2MzAwNzM0NQ==&mid=2247492269&idx=1&sn=4b346eb7ffb7b3a16a83071d46edeaa9&chksm=ce7d85b3f90a0ca516be74b6af8f53d6a182d9148265611837a865f965bdae4e2656aa3b1731#rd)
> **核心定位**：落點於「日频波段 × 监督回归 × 端到端表征」。解決傳統領漲-滯漲檢測依賴靜態線性相關圖譜的 prior gap，將「動態關係發現」與「橫截面收益預測」統一至可微框架。

**Status:** v0.5 — 基於 QuantML 導讀 + 原論文（如有）。benchmark 細節待升 v1。
**TL;DR:** ① 提出端到端框架，用稀疏交叉注意力動態識別領漲股及最優滯後期。② 核心 trick 是捨棄靜態圖譜，以可微 Top-k 機制學習資產對時變滯後值，預測端僅用滯後日原始價量特徵。③ 這對「端到端表征」軸★，因檢測模塊直接優化最終組合表現，打破檢測與預測的割裂。④ 導讀未給量化結果（僅提及平均每日約 10 bps 收益，超越 2-5 bps 成本）。

**X-Ray.** 將 DeltaLag 置於五軸 Pareto 前緣，其本質是「關係發現」的內生化。舊工程坑在於預計算相關圖譜的弱動量失效與跨資產信號的靜態錯配。DeltaLag 以注意力分數直接路由特徵，將非線性時變依賴轉化為可訓練的稀疏矩陣。預測端刻意降維至原始價量，犧牲部分序列表達力換取歸因透明度。其打不開的 envelope 在於極端流動性枯竭或同質化閃崩時的注意力坍縮（所有權重趨同）。對量化讀者意義：提供了一套「動態因子路由」的標準範式，證明簡單 MLP 配合精準的跨資產時序對齊，足以壓制黑箱序列模型。

## §1 · 架構 / Core Mechanism
**1.1 三大改動 vs 前作**
| 維度 | 傳統統計/靜態圖譜基線 | DeltaLag 改動 |
|---|---|---|
| 關係建模 | 預計算皮爾遜/距離相關（靜態線性） | 稀疏交叉注意力動態路由（時變非線性） |
| 訓練目標 | 獨立檢測 + 獨立預測（兩階段割裂） | 端到端聯合優化（檢測模塊直接服務組合收益） |
| 特徵輸入 | 依賴複雜時間嵌入或全量歷史 | 預測端僅用滯後日原始價量特徵（降維可解釋） |

**1.2 ⚡ Eureka** 用可微 Top-k 注意力分數直接錨定「誰領先、落後幾天」，將離散滯後選擇轉化為連續梯度流。
**1.3 信息流 ASCII**
```
[Target Stock] -> Query -> [Cross-Attention] <- Key (All Peers)
                                      |
                                 Top-k Routing
                                      |
[Lag Leaders @ t-lag] -> Raw Features -> Weighted Aggregation -> MLP -> R_{t+1}
```

## §2 · 數學層
📌 **Napkin Formula**：
$Q = W_q h_{t}^{target}$, $K = W_k [h_{t}^{peer}, ..., h_{t-T}^{peer}]$
$S = Q \cdot K^T \xrightarrow{\text{Top-k}} \{(i^*, \tau^*)\}$
$x_{agg} = \sum \sigma(S) \cdot x_{raw}^{(i^*, t-\tau^*)} \xrightarrow{MLP} \hat{r}_{t+1}$
**直覺**：注意力矩陣 $S$ 同時承擔「選股」與「選時」雙重職責，Top-k 操作雖不可微，但通過軟化/稀疏化技巧維持梯度流通。預測完全依賴跨資產信號，切斷目標股自身歷史的自迴歸依賴。
**Loss/訓練**：採用單調邏輯斯蒂回歸損失（基於 log-sum-exp 成對結構 + tanh 平滑），優化橫截面排序而非絕對回報值。訓練依 2010-2018/2019-2021/2022-2023 劃分，無複雜正則提及。

## §3 · 數據層
資料規模/頻率/市場/時段：美國股票日頻數據，2010-2023。劃分：訓練 2010-2018，驗證 2019-2021，測試 2022-2023。
股票池：S&P 500、NASDAQ、NYSE（剔除市值 <20 億美元）。
特徵：復權後 OHLC 比率、日收益率、對數交易量、日換手率。
樣本外與容量假設：導讀未明確披露樣本外容量上限或滑點模型，僅提及測試期平均每日約 10 bps 收益，远超 2-5 bps 典型成本。假設流動性可支撐 Top 10% Long / Bottom 10% Short 的橫截面換手。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations**：領漲-滯漲關係具「弱動量」特性，歷史統計無法捕捉動態非平穩性；模型依賴注意力機制，在極端行情下可能出現權重分佈異常。
**6.2 推斷的隱含假設**：Regime 依賴（訓練期 2010-2018 涵蓋特定宏觀週期，測試期 2022-2023 若遇結構性斷層可能失效）；容量假設（Top 10% Long/Short 組合在中小盤池可能面臨衝擊成本）；數據泄漏風險（預測使用當日特徵，需確保無未來函數；注意力計算依賴全市場截面，需嚴格平行處理）；Survivorship（導讀提及剔除市值 <20 億美元，但未明確說明是否處理退市股，存在潛在幸存者偏差）。

🎤 **Interview Tip** 
正確答：「DeltaLag 的核心不在於預測模型的複雜度，而在於將『領漲者識別』與『滯後期選擇』內生化為可訓練的注意力路由。預測端刻意使用原始特徵而非時間嵌入，是以可解釋性換取信號純度，避免黑箱模型對自身時間序列的過擬合。」
錯答：「它只是用 Transformer 做股票預測，效果因為 Attention 機制好所以 Sharpe 高。」（忽略跨資產對齊與排序損失的設計意圖）

**7.1 可證偽預測**：若 2026-06-30 前，該框架在納入真實滑點與 10 bps 交易成本後的實盤 Sharpe 跌破 1.0，或日均收益收斂至 <5 bps，則證明其動態路由在當前流動性結構下已失效。

## §8 · For the Reader
- **因子研究員**：將「可微 Top-k 路由」視為動態因子合成器，嘗試將傳統動量/反轉因子替換為原始價量，驗證注意力權重與因子輪動的相關性。
- **高頻執行**：關注 10 bps 收益與 2-5 bps 成本的邊際，實盤需精確建模盤中流動性與換手率，避免截面換手衝擊吞噬 Alpha。
- **組合配置**：利用模型自發湧現的聚類屬性（日均僅 38.18 只獨特領漲股），構建低相關性的領漲者組合，降低橫截面集中度風險。
- **LLM-agent / RL 策略**：將注意力分數作為環境狀態的稀疏表示，餵給 RL agent 進行動態倉位分配，替代靜態權重加權。
