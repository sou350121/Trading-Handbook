> **核心定位**：落點於「图关系×监督回归×端到端表征」軸，解決傳統分析師覆蓋動量策略依賴靜態加權與單跳聚合的工程坑，將多跳圖拓撲與非線性注意力機制引入日頻波段選股，證明結構先驗可顯著提煉領先-滯後溢出信號。

**Status:** v0.5 — 基於 QuantML 導讀 + 原論文（如有）。benchmark 細節待升 v1。
**TL;DR:** ① 將分析師共同覆蓋關係構建為動態圖，用 GAT 自適應聚合多跳鄰居動量特徵，替代傳統靜態加權。② 核心 trick 是注意力權重動態分配 + 端到端二分類預測（t+21），直接輸出超額/欠額表現概率。③ 對「图关系×监督回归」軸★，證明分析師網絡拓撲比純價格/行業/相關性矩陣更能捕捉信息傳播路徑。④ 實證年化收益 29.44%，夏普 4.06，波動僅 7%。

**X-Ray.** 在五軸 Pareto 中，本方法以「图关系」為結構錨點，用「监督回归」繞過傳統因子挖掘的線性假設，實現「端到端表征」的自動路由。它解了舊工程坑：分析師覆蓋信號長期被簡化為 1 跳鄰居加權平均，忽略多跳溢出與非線性權重動態。預測它打不開的 envelope：高頻摩擦環境（實盤成本未計）、結構性 regime shift（分析師覆蓋邏輯斷裂或流動性枯竭）、小盤/新興市場容量。對量化讀者的意義不在於 GAT 架構本身，而在於驗證了「機構信息中介網絡」作為圖先驗的 Alpha 提取效率；但高夏普/低回撤高度依賴無摩擦假設與大盤流動性溢價，實盤需嚴格壓力測試換手與滑點。

## §1 · 架構 / Core Mechanism
| 改動維度 | 前作/基線 (Analyst Matrix / NN) | GATanalysts |
|:---|:---|:---|
| **鄰居聚合** | 靜態加權平均 / 無圖拓撲 | 動態注意力權重 $\alpha_{ij}$ 自適應路由多跳鄰居 |
| **信號提取** | 線性疊加或獨立節點預測 | 端到端非線性映射 $GAT(A_t, X_t) \to Y_{t+21}$ |
| **特徵工程** | 單一動量視窗或手動因子 | 多視窗 log return (1,21,63,126,252d) + 短長比指標，堆疊為 8D 節點向量 |

⚡ **Eureka:** 注意力機制將「誰的動量更重要」從固定權重變為數據驅動的動態路由，讓模型自動學習分析師覆蓋網絡中的信息傳播路徑，跳過噪音節點，聚焦真正驅動溢出的關鍵鄰居。

## §2 · 數學層
📌 **Napkin Formula:** 
$h_i^{(l+1)} = \sigma\left(\sum_{j \in \mathcal{N}(i)} \alpha_{ij}^{(l)} W^{(l)} h_j^{(l)}\right)$, 其中 $\alpha_{ij} = \text{softmax}_j(\text{LeakyReLU}(a^T [W h_i || W h_j]))$。複雜度 $O(|E| \cdot D \cdot D')$。
**直覺:** 鄰居聚合不再是簡單平均，而是通過可學習的注意力係數動態分配權重，允許模型捕捉多跳溢出效應。
**Loss/訓練:** 二分類交叉熵（預測 t+21 超額/欠額概率）。Rolling window：每月重訓，Train 10天 / Val 10天 / Test 1天。Grid search 學習率、層數、注意力頭數、權重衰減。

## §5 · 評測 / Benchmark
| 數據集/市場 | Metric | 前SOTA | 本方法 | Δ |
|:---|:---|:---|:---|:---|
| S&P 500 (2006-2022) | Ann. Return | 未披露 | 29.44% | 未披露 |
| | Sharpe | 1.753 (NN) | 4.069 | 未披露 |
| | Vol | 8.32% (NN) | 7% | 未披露 |
| | Max DD 深度 | -6.42% (NN) | 未披露（導讀僅稱低於 NN） | 未披露 |
| | 回撤持續期占比 | 4.0% (NN) / 1.5% (GAT-1layer) | 1.0% | 未披露 |

**解讀:** Δ 主要來自注意力機制對多跳信息的非線性提純，而非槓桿或頻繁換手。但高夏普/低回撤在無成本假設下極易過擬合；實盤中換手與滑點將顯著壓縮 Δ。回撤持續期占比極低（1.0% 交易期處於回撤）暗示策略在趨勢行情中表現穩定，但對 regime shift（如流動性枯竭）的魯棒性未驗證。注意：導讀混用「回撤深度」與「回撤持續期占比」兩個指標，本表已分列，前者本方法值未披露。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations:** 未明確討論交易成本對實盤的具體影響（僅展示成本敏感性圖），未覆蓋小盤股/新興市場，未處理分析師覆蓋變動的生存者偏差。
**6.2 推斷的隱含假設:**
- **Regime 依賴:** 依賴分析師覆蓋網絡的結構穩定性（Jaccard >90%），若機構撤資或覆蓋邏輯改變，圖拓撲失效。
- **成本/容量:** 未披露換手率具體數值，僅知在 2bps 成本下仍正夏普，但實盤滑點/衝擊成本未計 `(未驗證)`。
- **數據泄漏:** 目標變量 t+21 為二分類閾值（Top/Bottom 25%），訓練時若使用全樣本統計量計算分位數，可能引入潛在的前瞻偏差。
- **Survivorship:** 僅 S&P 500，天然排除退市股，收益被高估。

## §8 · For the Reader
- **因子研究員:** 將分析師覆蓋矩陣視為「信息流動圖」，可嘗試將 GAT 的注意力權重 $\alpha_{ij}$ 提取為可解釋的領先-滯後因子，替代傳統截面動量。
- **高頻執行/組合配置:** 策略月頻調倉，但預測基於日頻特徵。實盤需關注每月第21天集中調倉的流動性衝擊，建議分倉執行或引入交易成本約束的優化器。
- **LLM-agent/RL 策略:** 可將 GAT 的圖結構作為 RL 的狀態空間先驗，用 PPO 學習動態持倉權重，替代硬編碼的 Top/Bottom 25% 截斷，提升風險調整收益。
- **研究學生:** 警惕「圖機器學習在金融中的過擬合陷阱」。建議復現時嚴格加入生存者偏差校正、滾動分位數計算與實盤成本模型，再評估 Δ 的真實性。
