# PDU 解構

> **發布**：2025-08-19 · KDD 2025
> **QuantML 導讀**：[KDD 25 | PDU: 如何利用“不确定性对比”发掘稳健超额收益](https://mp.weixin.qq.com/s?__biz=Mzg2MzAwNzM0NQ==&mid=2247491411&idx=1&sn=1645eed1869a707d31f7c3652edaf418&chksm=ce7e784df909f15ba86e3ba2763394d5d9ee0793ecd9e4e2981df2bcb277cc522970a2cec9d1#rd)
> **核心定位**：落點於「端到端表征 × 監督回歸」軸，解決了傳統量價模型在「時間-關係依賴緊耦合」與「市場不確定性風險」之間的 Prior Gap。

**五軸座標**

|:-:|:-:|:-:|:-:|:-:|
| `量价表格` | `日频波段` | `监督回归` | `端到端表征` | `全自动黑盒` |

**Status:** v0.5 — 基於 QuantML 導讀 + 原論文（如有）。benchmark 細節待升 v1。
**TL;DR:** ① 提出 PDU 模型，透過漸進式模組緊耦合時序、圖關係與多期動態，並引入 EMA 驅動的不確定性對比學習。② 核心 trick 是在加噪掩碼視圖下對齊確定性與不確定性表征，強制模型提取抗噪不變特徵。③ 這對「端到端表征」軸★ 的關鍵在於將對比學習從視覺/文本遷移至金融量價，解決了純監督回歸在市場波動下的表徵崩潰問題。④ 導讀未給量化結果。

**X-Ray.** PDU 的本質不是堆疊 RNN+GNN，而是將「依賴學習」與「風險對沖」在潛在空間內嵌。它解了兩個舊工程坑：一是時序與關係特徵簡單拼接導致的梯度衝突，二是靜態圖結構無法過濾高頻雜訊。但它的 Envelope 打不開：預定義關係圖（板塊/維基百科）是半靜態的，在 regime switch 或黑天鵝事件中會產生結構性滯後；對比學習雖提升魯棒性，卻以犧牲部分極端行情下的收益彈性為代價。對量化讀者而言，PDU 的價值不在於直接實盤，而在於提供了一套「不確定性感知」的表征正則化範式，可拆解為特徵篩選器或風險懲罰項，嵌入現有的多因子或強化學習框架。

## §1 · 架構 / Core Mechanism
| 改動維度 | 前作常見做法 | PDU 改動 | 工程意圖 |
|---|---|---|---|
| 時序依賴 | 標準 MHA / LSTM | 殘差注意力分數 (Residual Attention Scores) | 融合淺層與深層時序依賴，避免梯度消失 |
| 關係依賴 | 特徵拼接 / 靜態 GNN | GAT + 位置編碼 + 前饋網絡 | 捕捉時間感知的股票聯動效應，替代簡單拼接 |
| 多期動態 | 單一感受野卷積 | 多尺度卷積 (1x2, 1x3, 1x4) + 門控機制 | 並行捕獲短/中/長期動態，最大化歷史與未來趨勢互相關 |

**⚡ Eureka:** 用 EMA 維護目標網絡，在加噪掩碼的不確定性視圖下對齊原始視圖的表征，以非對稱更新穩定對比學習。
**信息流:**
```
Input(OHLCV+MA) → TDA(殘差MHA) → RDA(GAT+PosEmb) → MDC(多尺度卷積+門控) → PD_Rep
                                                      ↓
Uncertainty View(Noise+Mask) → PD_Rep_Noisy
                                                      ↓
EMA Contrastive Alignment → Ranking/Return Prediction
```

## §2 · 數學層
📌 **Napkin Formula:**
`L_total = L_point + L_rank + L_contrast`
`L_contrast = ‖z_clean - z_target‖²₂  (z_target 由 EMA 更新: θ_target ← τθ_target + (1-τ)θ_online)`
複雜度: O(N·T·d) 來自 MHA/GAT，多尺度卷積為 O(N·T·C) 常數級擴展。
直覺: 對比損失強制模型在輸入被高斯雜訊與隨機掩碼破壞時，仍輸出與原始數據一致的潛在表示，相當於在表征空間施加了強烈的平滑正則。
訓練細節: 僅在線網絡 (θ_online) 透過梯度下降更新，目標網絡 (θ_target) 僅透過 EMA 跟隨，避免優化競爭與表徵崩潰。總損失結合點態回歸、排序感知與對比損失進行端到端優化。

## §3 · 數據層
- **資料規模/頻率/市場**: 日頻 (Daily)。覆蓋 NASDAQ、NYSE、S&P 500、SZSE 四個市場。
- **特徵構造**: 日終 OHLCV + 5日、10日、20日、30日移動平均線回報率。
- **關係數據**: 板塊-行業關係 + 維基百科關聯（所有權、供應商-消費者），整合為鄰接矩陣構建靜態圖。
- **樣本外與容量假設**: 導讀未披露具體劃分比例與樣本量。容量受限於四大指數成分股，未涵蓋微盤股或跨境流動性異質性。

## §6 · 失效與隱含假設
**6.1 論文自述 limitations:** 依賴預定義的關係圖結構，未能動態適應市場結構的突變；對比學習機制增加了訓練複雜度與超參數調優成本。
**6.2 推斷的隱含假設:**
- **Regime 依賴**: 假設市場波動具有可對齊的統計穩定性。在極端流動性枯竭或政策干預 regime 下，靜態圖與對比對齊可能失效，導致排名相關性斷崖式下跌。
- **容量/成本**: 未建模交易成本。日頻波段策略在 SZSE/NASDAQ 的流動性與手續費結構差異巨大，實盤夏普比率可能遠低於回測。
- **數據泄漏**: 維基百科關聯與板塊分類若未嚴格按交易日切割，會引入隱性前瞻偏差；移動平均線回報率計算若未處理停牌/除權息，會產生幸存者偏差。

## §8 · For the Reader
- **因子研究員**: 提取 TDA 的殘差注意力分數與 MDC 的門控權重，作為動態特徵重要性因子，替代傳統的 IC/IR 加權。
- **高頻執行**: 本模型為日頻波段，不適用 HFT。但可將其「不確定性視圖」生成的表征方差，作為盤中流動性風險的預警信號。
- **組合配置**: 將 $L_{contrast}$ 的收斂程度轉化為組合層面的風險懲罰項，在優化器中動態調整高不確定性股票的權重上限。
- **LLM-agent / RL 策略**: PDU 的漸進式依賴結構可作為狀態編碼器 (State Encoder)，為 RL 交易 Agent 提供抗噪的市場狀態表征。
- **研究學生**: 重點復現 EMA 對比學習的參數更新邏輯與圖構建流程，這是可遷移至其他金融序列任務的通用 Trick。
